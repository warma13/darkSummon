## 二、世界观定位

### 2.1 血脉来源

绯夜是**纯系暗影血脉**的后裔，属于初裔神"**暗嗣玄渊**"一脉的直系血统。

但她的血脉谱系极为特殊——她的母亲是"冥契守护者"的最后一任祭司，父亲则是"深渊饮者"的叛逃者。两股本应对立的暗影血脉在她体内交汇，产生了一种罕见的血脉共振状态：她既能像冥契守护者一样精准感知暗影的秩序与流向，又继承了深渊饮者将暗影能量极限压缩的天赋。

这种双重血脉的交汇使她的暗影能量密度远超同族——同样一击，她释放的暗影不是"面"，而是被压缩到极限的"点"。这就是她被称为"暗影之针"的原因。

### 2.2 身世背景

绯夜出生于暗影界边境的"黄昏裂隙"——光界与暗影界交界最动荡的区域。这里的暗影能量被光系能量反复灼烧和净化，导致当地暗影生灵极度稀少，而存活下来的个体都拥有异常凝练的暗影核心。

她的母亲在她出生不久后就因血脉枯竭而消散——这是冥契守护者祭司的宿命，祭司的生命力会持续流向她们所维护的暗影屏障。绯夜被父亲独自养大，在黄昏裂隙中学会了一项至关重要的生存法则：

**"暗影越稀薄的地方，越要把每一缕暗影用在刀刃上。"**

这塑造了她的战斗哲学——不浪费一丝能量在无意义的范围攻击上，所有力量都精准聚焦于一个目标。

十六岁时，父亲被一支光誓者教会的远征队捕杀。绯夜目睹了父亲的消亡，也在那一刻完成了暗影血脉的第一次觉醒——她以一道纯粹的暗影刺穿了远征队队长的护盾，这道暗影细如针尖，却穿透了七层光系屏障。

此后她独自游历艾索斯大陆，被召唤师联盟发现并招入。她对"正义"和"邪恶"不感兴趣——她只忠于战斗本身的纯粹与精准。

### 2.3 与其他角色的关系

| 角色 | 关系 |
|------|------|
| 暗影君主（主角） | 绯夜是少数能让暗影君主感到"同源共振"的存在。她的暗影血脉纯度之高，甚至在某些瞬间让暗影君主误以为感知到了玄渊的气息。彼此之间保持着沉默的尊重。 |
| 翎嫣 | 暗影与自然的天然对极。翎嫣的自然之力会让绯夜微微不适（"太亮了"），但绯夜承认翎嫣的增益能让她的暗影更凝练——两人在战场上是意外地高效的搭档。 |
| 虚空龙王 | 绯夜对虚空龙王的链式攻击嗤之以鼻——"把力量分散给四个目标？浪费。"但她暗中认可龙王对BOSS的执着。 |
| 堕天使长 | 堕天使长的增伤标记是绯夜最喜欢的战场条件之一。被标记的目标在她的暗影之针下几乎没有生还可能。 |

## 三、外形设计

### 3.1 整体风格

暗影血脉的典型特征与极致战斗美学的融合——一切服饰与形态都服务于"精准"和"锋利"两个关键词。

### 3.2 详细描述

| 特征 | 描述 |
|------|------|
| 身高 | 168cm，身形修长但并非纤弱，有明确的肌肉线条感（长年战斗的痕迹） |
| 肤色 | 苍白冷调，带极淡的银灰色底色，如同月光下的大理石。不是病态的苍白，而是暗影血脉天然的光反射——在暗处时皮肤会透出极淡的深紫色内光 |
| 发色 | **鸦羽黑**，在暗影能量激活时发梢渐变为**深绯红**（如同夜空边缘的晚霞），这也是她名字"绯夜"的由来。头发长度及腰，但战斗时以暗影能量束缚成紧致的高马尾 |
| 瞳孔 | **深红色**，是玄渊一脉最纯正的血脉标志。瞳孔中有细微的暗影纹路流动，在锁定目标时瞳孔会短暂收缩为竖瞳状——这是深渊饮者血脉的特征 |
| 血脉纹路 | 深紫色暗影纹路从左手手背沿手臂蔓延至左肩与颈侧，纹路细密如同裂纹。平时隐现，战斗时亮起绯红色光芒 |
| 服饰 | 合身的暗黑色作战服，材质是暗影蛛丝（暗影界特产，极轻极韧），露出左臂（展示血脉纹路）。腰间别着一柄短刃——但她几乎不用，真正的武器是暗影本身。足部是暗色长靴，靴跟有隐约的暗影能量流动 |
| 标志物 | 左耳佩戴一枚**绯红色水滴形耳坠**，是母亲的遗物，内封着最后一缕冥契守护者的暗影能量。绯夜在战斗中能从中汲取微量能量，这也是她暗影凝练度异常高的原因之一 |

### 3.3 战斗形态

当绯夜进入全力状态，血脉纹路全部亮起，发色从黑渐变为深绯红，身后凝聚出一道模糊的暗影轮廓——那是玄渊一脉深渊饮者的血脉幻象，形似一只展翼的暗影鸦。这个形态下她的每一次攻击都会在目标身上留下绯红色的暗影裂痕。

## 四、性格核心

### 4.1 关键特质

- **沉默寡言但并非冷漠**：绯夜极少说话，不是因为冷酷，而是因为她认为"语言是不精确的"——就像散射的范围攻击一样浪费。但一旦她开口，每句话都直指要害。
- **极致的专注力**：无论战斗还是日常，她做任何事都保持着猎手般的专注。吃饭时不会同时说话，观察敌人时能保持几小时一动不动。
- **以"精准"为最高美学**：她不认为力量大就是强，不认为杀得多就是赢。一击必杀——用最少的暗影能量造成最大的伤害，这才是她追求的完美。
- **隐藏的温柔**：虽然外表冷厉，但她会无声地照料受伤的同伴，会在夜间为营地编织暗影屏障（不告诉任何人）。她继承了母亲冥契守护者的本能——守护，只是方式不同。

### 4.2 标志性语录

> "一击。够了。"

> "你在浪费暗影。那些飞散出去的能量，原本可以让你多杀一个敌人。"

> "我母亲用一生守护暗影界的屏障。我用每一击守护她留下的遗愿——让暗影成为保护，而不是恐惧。"

> "你想了解我？……不必。你只需要知道，站在我身后是安全的。"

## 六、技能设计

### 6.1 技能一览

| 槽位 | ID | 名称 | 类型 | 解锁 | 核心效果 |
|------|-----|------|------|------|---------|
| 技能1 | `shadow_needle` | 暗影之针 | 被动 | Lv100 | 普攻叠加暗影印记，满层触发穿刺爆发 |
| 技能2 | `blood_eye` | 绯瞳锁定 | 被动 | Lv500 | 暴击体系，持续攻击同一目标逐步提升暴击率 |
| 技能3 | `abyss_strike` | 深渊一刺 | 主动 | Lv1500 | 对当前目标释放极高倍率的单体爆发 |

### 6.2 技能详细设计

---

#### 技能1：暗影之针（Shadow Needle）

**类型**：被动（Lv100 解锁）
**升级方式**：随升星提升（6节点：★5/10/15/20/25/30）

**机制描述**：
绯夜的每次普通攻击会在目标身上叠加1层**暗影印记**（最多5层，持续4秒，刷新计时）。当印记达到5层时，自动触发**穿刺爆发**：消耗全部印记，对目标造成 ATK × 200% 的额外暗影伤害。

**穿刺爆发的伤害无视目标 20% 护甲**（体现暗影压缩穿透的设定）。

**数值**：

| 参数 | 基础值（★0） | 满星值（★30，×9.88） | 说明 |
|------|------------|-------------------|------|
| 每层印记数 | 1层/次攻击 | 1层/次攻击 | 不缩放 |
| 最大层数 | 5层 | 5层 | 不缩放 |
| 印记持续 | 4秒 | 4秒 | 不缩放 |
| 穿刺倍率 | ATK × 200% | ATK × 1976% | 核心缩放参数 |
| 护甲穿透 | 20% | 20% | 不缩放（已经很强） |

**Config 定义**：
```lua
{ id = "shadow_needle", name = "暗影之针",
  desc = "普攻叠加暗影印记（最多5层，持续4秒）；满5层触发穿刺爆发，造成ATK×200%暗影伤害，无视20%护甲",
  type = "passive",
  maxStacks = 5, stackDuration = 4.0,
  burstAtkPct = 2.0,
  armorIgnore = 0.20,
  starScale = true },
```

**设计理念**：
- 5层叠满需要约4.5秒（0.9秒攻速×5次），配合持续4秒的印记窗口，要求绯夜持续攻击同一目标才能触发
- 完美契合"单体输出"定位：攻击分散的玩法会让印记过期，奖励专注单目标的策略
- 护甲穿透体现了"暗影压缩穿透"的世界观设定

---

#### 技能2：绯瞳锁定（Blood Eye Lock）

**类型**：被动（Lv500 解锁）
**升级方式**：随升星提升

**机制描述**：
绯夜连续攻击同一目标时，每次攻击提升 3% 暴击率（最高叠加10次，+30%暴击率），暴击伤害固定为 +50%。切换目标或目标死亡时，暴击率重置为0。

目标死亡时，绯夜获得 2 秒"**余韵**"状态：下一个目标直接继承前一个目标 50% 的暴击率层数（向下取整），帮助她在换目标时不至于完全损失积累。

**数值**：

| 参数 | 基础值（★0） | 满星值（★30，×9.88） | 说明 |
|------|------------|-------------------|------|
| 每次暴击率增量 | 3% | 29.6% | 核心缩放参数 |
| 最大叠加次数 | 10次 | 10次 | 不缩放 |
| 暴击伤害加成 | +50% | +50% | 不缩放（避免过于膨胀） |
| 余韵继承率 | 50% | 50% | 不缩放 |
| 余韵持续 | 2秒 | 2秒 | 不缩放 |

> 满星时暴击率上限为10×29.6%=296%（实际cap在100%），意味着只需3-4次攻击即可100%暴击，极大提升了换目标后的恢复速度。

**Config 定义**：
```lua
{ id = "blood_eye", name = "绯瞳锁定",
  desc = "连续攻击同一目标，每次+3%暴击率（最多+30%），暴击伤害+50%；目标死亡后2秒内，新目标继承50%暴击层数",
  type = "passive",
  critRatePerHit = 0.03, maxCritStacks = 10,
  critDmgBonus = 0.50,
  echoRatio = 0.50, echoDuration = 2.0,
  starScale = true },
```

**设计理念**：
- 持续打同一目标越久，暴击率越高，与技能1的印记机制形成双重奖励
- "余韵"机制避免了目标死亡后完全归零的挫败感，也体现了"暗影记忆目标气息"的设定
- 暴击伤害固定+50%不缩放，避免后期暴击伤害过度膨胀

---

#### 技能3：深渊一刺（Abyss Strike）

**类型**：主动（Lv1500 解锁，CD 14秒）
**升级方式**：随进阶提升（4节点：阶5/10/15/20）

**机制描述**：
绯夜凝聚全部暗影能量，对当前攻击目标释放一记极致穿刺——造成 ATK × 800% 暗影伤害。

**额外效果**：
- 若目标当前身上有暗影印记，每层印记额外增加 100% ATK 伤害（5层满印记 = 额外500%ATK）
- 释放后消耗所有印记层数
- 此次攻击必定暴击（受绯瞳锁定的暴击伤害加成影响）

**数值**：

| 参数 | 基础值（阶0） | 满阶值（阶20，×5.07） | 说明 |
|------|------------|-------------------|------|
| 基础倍率 | ATK × 800% | ATK × 4056% | 核心缩放参数 |
| 每层印记额外 | ATK × 100% | ATK × 507% | 随主动等级缩放 |
| 冷却时间 | 14秒 | 9.66秒 | CD ×0.69 |
| 必定暴击 | 是 | 是 | 不变 |

> 满养成极限一击（满阶+满5层印记）：4056% + 507%×5 = **6591% ATK**，再乘暴击1.5倍 = **约9887% ATK 单次伤害**。

**Config 定义**：
```lua
{ id = "abyss_strike", name = "深渊一刺",
  desc = "对当前目标造成ATK×800%暗影伤害，必定暴击；每层暗影印记额外+100%ATK伤害并消耗印记",
  type = "active", interval = 14,
  baseAtkPct = 8.0,
  stackBonusPct = 1.0,
  guaranteedCrit = true,
  starScale = true },
```

**设计理念**：
- 核心爆发技能，与技能1的印记叠加形成完美的"蓄力→爆发"循环
- 必定暴击确保这一击的伤害天花板（配合技能2的+50%暴击伤害）
- 消耗印记意味着"深渊一刺"后需要重新叠层，创造了张弛有节的输出节奏
- 14秒CD恰好与5层印记的叠满时间（约4.5秒）形成节奏：叠满→爆发→CD期间重新叠层→再爆发

---

### 6.3 技能连锁设计（战斗循环）

```
普攻×5（约4.5秒）→ 暗影印记叠满5层
                    ↓
               绯瞳暴击率逐步攀升
                    ↓
              深渊一刺 释放！
              （800%+500% ATK，必定暴击）
                    ↓
              印记清零，暴击率保留
                    ↓
              继续普攻叠层（CD 14秒）
                    ↓
              约9.5秒后印记再次叠满
                    ↓
              深渊一刺 CD 好了 → 再次爆发
```

**输出节奏**：约每14秒一次大爆发，期间持续输出（暴击率不断攀升）。

## 八、阵容适配分析

### 8.1 最佳搭配

| 搭档 | 原因 |
|------|------|
| **堕天使长** | 增伤标记（+20%受伤）→ 绯夜的深渊一刺伤害直接翻倍收益 |
| **翎嫣** | 自然馈赠的攻击力加成 → 直接提升绯夜所有技能的ATK基数 |
| **幽魂刺客** | 致命标记（+12%受伤）→ 进一步放大单体伤害 |
| **破甲骑士** | 护甲削减 → 与暗影之针的20%穿甲叠加，高甲目标克星 |

### 8.2 阵营协同

归属 undead 阵营，与骷髅小兵、骷髅弓手、幽魂刺客、暗影法师、死灵术士组成亡灵阵容：
- 2亡灵：+8%攻击
- 3亡灵：+15%攻击，+8%攻速
- 4亡灵：+25%攻击，+15%攻速，触发"亡灵诅咒"（敌人死亡AOE 5%ATK）

### 8.3 定位对比

| 英雄 | 定位 | 对单DPS | 对群DPS | 特点 |
|------|------|---------|---------|------|
| **绯夜** | 纯单体 | **极高** | 低 | 叠层爆发，越打越强 |

## 十一、设计总结

绯夜填补了常驻UR池中**纯单体高爆发输出**的空缺：

- **世界观融合**：纯系暗影血脉（冥契守护者×深渊饮者的混血），"暗影压缩穿透"的战斗理念完美契合单体输出定位
- **机制独特性**：印记叠层→爆发消耗的循环，区别于现有所有英雄的战斗模式
- **养成深度**：三个技能互相联动（印记→暴击→爆发），升星/进阶带来清晰的伤害成长感
- **阵容价值**：undead阵营补强，与堕天使长/翎嫣等现有UR形成互补而非竞争
