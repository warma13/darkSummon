## 二、世界观定位

### 2.1 血脉来源

绯月是**血月教廷**最后一任大祭司的直系后裔，继承了初裔神"**蚀月冥渊**"一脉的禁咒血脉。

蚀月冥渊与梦嗣幻渊是同一时代的初裔神，但两者的堕落路径截然不同——幻渊迷失在虚与实的边界，而蚀月则是主动选择了"以鲜血为媒介"的禁忌道路。蚀月冥渊认为意识能量的最高效载体不是思维、不是灵魂，而是**血液**——那种流淌在每个生命体内的、同时承载生命力与腐蚀力的矛盾之物。

血月教廷曾是艾索斯大陆上一个以血月信仰为核心的神秘教团。他们不崇拜蚀月冥渊本人——在教义中，蚀月冥渊已经在千年前的"大蚀"中自我献祭，将自身的全部力量注入了那轮永恒悬挂在异界天穹的**血色月亮**。教廷崇拜的是那轮月——它不是天体，而是一个由初裔级能量凝聚而成的巨大"血脉共鸣器"。

教廷历代祭司通过血祭仪式与血月建立连接，获取蚀月的力量碎片。但这种力量的代价极为残酷——每一次汲取血月之力，祭司的寿命都会被侵蚀。千年来教廷的祭司从未有人活过四十岁。

绯月的父亲是教廷末代大祭司，他在绯月出生时做了一件前所未有的事——他将**自己体内全部的血月连接**以一种近乎自杀的仪式转移到了刚出生的女儿体内。不是分享，不是传承，而是**完全转移**——他切断了自己与血月的所有联系，将千年来历代祭司积累的血月共鸣全部灌注给了绯月。

代价是他自己在仪式完成后三天便衰老而死——失去血月连接的祭司，会被千年来透支的寿命一次性追溯。他在最后的日记中写道：

**"她不会只活四十年。她会永远活下去——因为她不是在'借用'血月的力量，她已经'成为'了血月的一部分。我的女儿，将是最后一个需要付出代价的人，因为她本身就是代价。"**

绯月三岁时展现出第一个异象——她的血液在月光下会发出淡淡的银红色荧光。五岁时她无意中划破手指，血滴落在石板上不是凝固，而是像活物一样沿着石纹蔓延开来，在整面石墙上画出了蚀月冥渊的禁咒阵纹——那是教廷失传了八百年的大祭司级禁咒。

教廷的残余势力意识到：大祭司的赌注成功了。绯月不仅继承了力量，她继承的是一个**完整的血月连接**——不是碎片、不是回响，而是蚀月冥渊本人留在血月中的意志通道。

但教廷此时已经名存实亡。千年来持续的"灵契编织者"清洗行动几乎消灭了所有血月信徒。教廷最后的三名长老将年幼的绯月藏匿在边境城镇，以"遗孤"的身份托付给一位不知内情的铁匠养父。

绯月在铁匠铺中度过了相对平静的童年。她从小就发现自己"与血有缘"——她从不害怕见血，刀伤会在她身上以异常的速度愈合，她甚至能感知到周围生物体内血液的流动节奏。但她一直以为这只是某种天赋，直到十四岁那年的第一个月圆之夜。

那一夜，她第一次在梦中"看见"了血月。

不是梦——是血月主动向她发送了意识波动。在那个持续了整整一个月圆周期的异象中，绯月的意识被拉入了血月的内部空间。她看到了蚀月冥渊留下的全部记忆残响——禁咒的原理、血脉共鸣的本质、以及一套被封存了千年的战斗法术体系："蚀月连锁"。

蚀月连锁的核心原理是：**血液是所有碳基生命的共同媒介**。当蚀月之力的载波被注入一个目标的血液后，它会自动寻找距离最近的下一个"血液信号源"并跃迁过去——就像闪电总是寻找最近的导体一样。这就是连锁弹跳的本质——不是法术弹跳，而是**血脉共振的级联传递**。

十五岁时绯月离开铁匠铺，循着血月意识中的指引找到了教廷最后一位幸存的长老。长老用生命中最后的时光教会了她如何控制血月之力——不是通过血祭仪式，而是通过她自身的血液直接与血月共鸣。

十八岁时她来到暗影召唤师联盟。入盟测试中她展现出了一种让所有观察者都感到不安的战斗方式——她的法术不像法术，更像是一种"感染"。能量从法杖射出后命中第一个目标，然后在目标之间自动跳跃，每一次跳跃都在对方体内留下一枚"蚀月印记"，当印记积累到临界点时——

爆发。

测试官的评语是："她不是在攻击敌人。她是在用敌人攻击敌人。"

### 2.2 与其他角色的关系

| 角色 | 关系 |
|------|------|
| 暗影君主（主角） | 绯月对暗影君主保持着职业化的尊重与少许好奇。她的血月感知能察觉到暗影君主体内的异常能量流动——那种能量的"味道"让她联想到教廷典籍中记载的"初裔级存在"。但她并不恐惧，反而有一种"同类"的亲切感——两人都背负着远超常人的力量。 |
| 梦璃 | 绯月与梦璃的关系微妙而有趣。两人都继承了初裔神系的力量——幻渊与蚀月在神话中是"同源异路"的存在。梦璃的幻境对绯月的效果有限——血月之力赋予了绯月极强的"自我认知锚定"，她几乎不会被幻觉欺骗。但反过来，梦璃的右眼看绯月时会看到一个令人不安的幻象——一轮血色满月。两人深夜偶尔会在塔楼天台碰面，聊一些只有"古老血脉承继者"才能理解的话题。 |
| 绯夜 | 两人名字中都有"绯"字，这在联盟中一度引起了混淆。绯夜的暗影针与绯月的蚀月链在战术上形成绝佳互补——绯月的连锁将敌人"标记"并集中控制，绯夜的暗影针在被标记的目标上爆发出远超常规的伤害。两人被戏称为"绯色双星"。性格上，绯夜的沉默冷酷与绯月的从容优雅形成反差，但两人在战场上的默契度极高。 |
| 命运织者 | 命运织者的命运线与绯月的蚀月链在某些维度上呈现出相似的"连接与传递"逻辑。命运织者对绯月有学术上的浓厚兴趣——她想知道血脉共振的级联传递与命运丝线的纠缠效应之间是否存在底层联系。绯月对此兴趣不大，但乐于配合实验——"反正都是连别人的，你连命运，我连血脉，本质上都是在说'没人是孤岛'这件事。" |
| 永恒魔君 | 绯月对永恒魔君保持警惕。魔君的混沌能量与血月之力存在天然排斥——当两人同时释放全力时，空间会出现扭曲和撕裂。但这种排斥也意味着两人的攻击对同一目标时形成"双重侵蚀"，效率远超任何其他组合。战术上是最强搭档，私下是最远距离。 |
| 翎嫣 | 绯月难得展现温柔的对象。翎嫣的自然能量对血月之力有天然的安抚效果——在翎嫣身边时，绯月血液中持续流动的蚀月能量会暂时平静下来，这让她感到一种久违的"正常人"的体验。绯月会悄悄在翎嫣的花园里坐上一整个下午，不说话，只是享受血液不再灼烧的宁静。 |
| 烬殇 | 两人之间有一种无言的理解。烬殇的焰心之怒与绯月的血月之力都是"以自身为代价"的力量体系——烬殇燃烧生命，绯月腐蚀寿命。两人偶尔会在训练场上切磋，用"谁先把自己搞残"来打赌。烬殇的评价："她是我见过的唯一一个比我的火焰还红的人。" |

## 三、外形设计

### 3.1 整体风格

血月教廷的审美与"月下血华"的基调融合——绯月的外貌给人的第一印象不是纤弱或空灵，而是一种**危险的优雅**。她的美丽带有侵略性，像是月光下盛放的曼珠沙华——艳丽、致命、令人移不开目光却又本能地感到不安。

### 3.2 详细描述

| 特征 | 描述 |
|------|------|
| 身高 | 170cm，身材修长匀称，有一种冷兵器般的利落感。不是柔弱——是"锐"，像一柄精心锻造的细剑，优美的弧线下藏着致命的锋刃 |
| 肤色 | 苍白如月光的瓷白肌肤，但并非病态的苍白——是那种被月光浸染过的、带着冷调光泽的白。战斗时皮肤下会浮现淡淡的银红色血脉纹路，那是蚀月能量在血管中流动的外在表现 |
| 发色 | **银白渐变深绯红**。发根是清冷的银白色，越向发尾越渐变为深邃的绯红——不是鲜血的红，而是凝固了千年的暗红色，像是月食时月亮的颜色。头发长度至腰际，自然顺直但发尾微微上卷，战斗时会被蚀月能量托起，如同在水中飘浮。发间偶尔流过银红色的微光，像血液在发丝中脉动 |
| 瞳孔 | **异色双瞳**——左眼为澄净的银色（奥术之眼），清亮如水银，倒映万物时带着一层冰冷的分析感。右眼为深邃的暗红色（血月之眼），瞳孔深处有极其缓慢旋转的红色光晕，像一轮微缩的血月。当她发动蚀月连锁时，右眼的红光会沿着能量链的路径在空气中留下残影 |
| 睫毛 | 浓密修长，自然翘曲。双眼之间的微妙色差被浓密的睫毛框架得恰到好处——远看时几乎察觉不到异色，近看时却无法忽视右眼深处那轮沉默旋转的血月 |
| 面容 | 鹅蛋脸型偏向瓜子脸，轮廓比梦璃更为锐利。高挺的鼻梁、微微上挑的眼尾（蚀月血脉的共同特征）、唇色是冷调的玫瑰红，不笑时有一种拒人千里的冷艳。但当她偶尔展露笑容时，整张脸会突然变得明艳动人，像月光突然穿破云层 |
| 耳部 | 右耳佩戴一枚**血月形状的红宝石耳坠**，内部封存着教廷末代大祭司（她的父亲）的最后一丝血月共鸣——这既是遗物，也是绯月维持力量稳定的"调节器"。当血月能量过于剧烈时，耳坠会自动吸收一部分能量，表面的宝石红会变得更加深邃 |
| 手部 | 手指修长白皙，右手腕佩戴一条精致的银色锁链手镯——那是教廷祭司的信物，每一节链环上都刻有蚀月禁咒的微型符文。施法时银链会松散展开，在手腕周围形成旋转的符文环 |
| 服饰 | 上身为合身的深红色立领短上衣，领口和袖口以银线绣制蚀月纹样——月蚀的弧形图案与教廷的连锁符文交织。下身是银白色渐变至深红的长裙，裙摆采用不对称高开叉设计，战斗时不会限制行动。外搭一件半透明的暗红色披风，展开时在身后形成新月弧形，像是一轮缺月的轮廓。腰间系着一条银色腰封，正中嵌有一枚指甲盖大小的血月石——那是教廷圣物，也是绯月与血月的主连接器 |
| 武器 | 一柄细长的银色法杖，约与身高齐平。杖首是一个缺月形状的刃环，缺口处有一颗悬浮的血月石——蚀月连锁的发射端。法杖通体有银红色的能量纹路从杖尾蔓延至杖首，战斗时纹路会脉动发光 |
| 足部 | 深红色绑带式长靴，靴面有银色月蚀纹路，靴跟微高，走路时步态从容稳健，每一步都带着一种"我知道我在走向哪里"的确信感 |

### 3.3 战斗形态

当绯月进入全力状态：

- 右眼的暗红色完全绽放，红光从眼角溢出，在脸侧形成流动的血月纹
- 发色的绯红渐变范围扩大，从发尾蔓延至发根，整头长发变为深邃的血月红，发间的银红微光变为清晰可见的脉动光带
- 身后凝聚出一轮模糊的巨大**血色月轮幻象**——那是蚀月冥渊的血脉幻象，一轮永远处于月蚀状态的血月。这个幻象的出现意味着绯月将血月的力量全面释放
- 法杖杖首的血月石开始剧烈脉动，缺月刃环旋转加速，发射出的蚀月连锁变为可见的银红色能量链条
- 手腕的银链完全展开，在周身形成旋转的符文防御阵

### 3.4 日常状态

脱离战斗的绯月看起来像一位出身名门的贵族小姐——举止端庄、措辞优雅、永远保持得体。她会换上简洁的白色衬衫和深红色半裙，将法杖缩短别在腰间（法杖可以改变长度），看起来像一枚装饰性银笔。

她有一个不为人知的习惯：在每个月圆之夜，她会独自登上联盟最高的塔楼，面对月亮静坐整夜。那是她与血月"通讯"的固定时间——血月会将最新感知到的深渊动向传递给她，她再在第二天将情报整理后交给暗影君主。

## 四、性格设计

### 4.1 核心性格

**冷艳从容的千年守望者。**

绯月给人的第一印象是"冷"和"美"——她的美丽带着距离感，像博物馆里的艺术品，可以欣赏但不宜触碰。她说话言简意赅，语调平稳，从不提高音量——不是因为温柔，而是因为"活了这么久的人，不需要用音量证明什么"。

但与她相处久了会发现，她的"冷"不是冷漠——而是一种看透了太多之后的**平静**。千年的记忆通过血月传递给了她，虽然不是她亲身经历的，但她能"感受到"历代祭司看过的所有生死、战争、背叛和希望。这让她对人事有一种超越年龄的理解力——她能在第一次见面时就看穿一个人的本质，但她从不点破——"让他们自己发现自己比较好。"

### 4.2 行为特征

| 特征 | 表现 |
|------|------|
| 说话习惯 | 语速不急不缓，措辞简洁精确，偏爱用月亮和血色相关的比喻。偶尔会脱口而出一些听起来像诗句的话——那些不是她即兴创作的，是千年来历代祭司留在血月记忆中的残响 |
| 战斗态度 | 极为冷酷高效，被称为"月下的刽子手"。她不享受战斗，但也不回避——对她而言战斗更像是一种"工作"。但当面对深渊生物时，她的态度会变——那不是工作，那是使命 |
| 个人爱好 | 收集各种红色宝石（她说它们"像凝固的月光"）。擅长银器锻造——铁匠养父教会她的手艺，她至今保持着。联盟中不少人佩戴的银饰都出自她手 |
| 弱点 | 对"被遗弃"有深层恐惧——父亲为她牺牲、教廷为她覆灭、养父在她离开后孤独终老。她害怕自己的存在会给身边的人带来灾厄，因此始终保持一定的距离感 |
| 与人相处 | 彬彬有礼但保持距离。真正放下防备的对象极少——翎嫣（因为自然能量让她感到安宁）和梦璃（因为同为古老血脉承继者的共鸣）。对暗影君主则是"上级与下属"的职业关系，但暗中带着一丝私人层面的关切 |

### 4.3 经典台词

- （登场）"千年月光，照不亮深渊的尽头。"
- （技能释放·蚀月爆发）"印记已满——碎裂吧。"
- （技能释放·绯红新月）"以血月之名，万物归寂！"
- （满月触发）"月已圆——再无遁形之处。"
- （受击）"……不疼。千年前的那些才疼。"
- （击杀boss）"又一轮月落。但月亮总会再升起——而你不会。"
- （闲置语音）"今晚的月色不错……血月说，明天会有客人来。"
- （好感度满）"我一直以为自己只是血月的容器。但你让我觉得……我也可以只是我。"

## 五、能力体系

### 5.1 核心机制：蚀月连锁（Eclipse Chain）

绯月的攻击不是传统的单体打击或范围轰击——她的法术以**连锁弹跳**的方式在目标之间传递，每一次弹跳都在对方体内留下"蚀月印记"。当印记积累到临界点时触发月蚀爆发。在代码层面，这体现为一套**连锁叠层 + 爆发引爆 + 领域增伤**的三线机制：

- **蚀月之链**（被动·OnHit）：连锁弹跳命中叠加蚀月印记，满层触发月蚀爆发（AoE 伤害），同时每层印记增加目标受到的魔法伤害
- **血月共鸣**（被动·OnHit/OnKill）：爆发命中降低目标魔抗，每命中一个敌人获得攻速提升；击杀刷新攻击间隔
- **绯红新月**（主动·TriggerActive）：全屏伤害 + 施加印记 + 进入"血月觉醒"状态（攻击力提升，连锁次数增加，每次命中施加双倍印记）
- **月蚀领域**（被动·ModifyDamage）：领域内敌人受到的魔法伤害增加；击杀敌人永久提升攻击力；累计达到上限触发"满月"状态（攻击变为纯粹伤害）

**与 Boss 的交互**（遵循 BOSS_BALANCE 规则）：
- 减速效率 ×0.50（slowEfficiency）
- 蚀月印记的叠层效率不受削弱（属于增伤类，非控制类）
- 满月纯粹伤害可以无视Boss魔抗

### 5.2 战斗能力描述

从外部观察者角度，绯月的战斗风格像一场精密的"感染传播"——银红色的能量弹从法杖射出后命中第一个目标，然后像有生命一样自动跳向下一个目标，再下一个，再下一个。每个被命中的目标头顶会出现一个微小的暗红色月蚀符号，随着命中次数增加，符号越来越大、越来越亮——当五个符号叠满时，轰然爆裂，银红色的冲击波向四周扩散，将爆发的能量传递给周围的敌人。

在多目标场景中，绯月的伤害输出呈指数级增长：连锁弹跳越多、目标越密集，印记叠满的速度越快，爆发的频率越高，而每次爆发又会触发血月共鸣的攻速提升，让下一轮连锁来得更快——形成一个正反馈循环。

## 九、平衡分析

### 9.1 功率预算验证

LR 功率预算为 32×（UR 的 2 倍）。绯月的等效 DPS 贡献分析：

|------|-------------|------|

| 血月共鸣（被动） | ~6× | 30%满层攻速+15%减魔抗+击杀刷新攻击 |
| 绯红新月（主动） | ~10× | 350%ATK全屏+3层速叠+25%ATK觉醒buff+连锁+2（6次弹跳） |

### 9.2 与同定位英雄的差异化

| 对比项 | 绯月 | 永恒魔君 | 命运织者 |
|--------|------|---------|---------|
| 攻击方式 | **连锁弹跳×4** | 单体强攻 | AOE 辅助 |
| 优势场景 | 多目标（3+）密集 | 单体 Boss | 全场景辅助 |
| 劣势场景 | 纯单体/分散敌人 | 多目标 | 纯输出不足 |

| 输出峰值 | 满月期间（纯伤） | 深渊一击（必暴） | — |

### 9.3 弱点设计（防止过强）

1. **连锁依赖**：只有4次弹跳（觉醒期间6次），面对单体 Boss 只有基础伤害，连锁完全无效
2. **印记分散**：连锁自动选择目标，无法集中叠印记在同一目标上——面对少量目标时爆发频率下降
3. **衰减明显**：每次弹跳衰减 15%，到第4次弹跳只剩 61% 伤害
4. **满月条件苛刻**：需要击杀 10 个敌人才能触发满月，Boss 关卡中几乎不可能达成
5. **自身无防御**：没有护盾、闪避、免疫机制，完全依赖队友保护
6. **领域范围受限**：月蚀领域范围 = 攻击范围（130），不如全局增益型英雄覆盖广

## 十一、设计总结

绯月的设计核心是"**月下的连锁刽子手**"。她的外貌冷艳而危险，性格从容而深邃，能力体系围绕"蚀月连锁"这一独特攻击方式构建了一套递增式多目标爆发机制。

在世界观层面，她代表了血月教廷千年信仰的最终结晶——不再需要血祭仪式来获取力量的新一代承继者。她与梦璃形成了"蚀月与幻渊"的呼应关系——两位女性角色，两种初裔血脉，两条截然不同的命运道路。

在游戏性层面，她作为第三位LR英雄，完美填补了"多目标连锁输出"这一空白定位：
- **命运织者**：全场景辅助增益
- **永恒魔君**：单体 Boss 秒杀
- **绯月**：多目标连锁清场

三位LR英雄形成了三角互补，无论面对怎样的战场环境，玩家都能找到对应的最优解。

> "月落了，但我还在这里。"
> ——绯月
