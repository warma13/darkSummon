## 二、世界观定位

### 2.1 血脉来源

梦璃是**纯系幻梦血脉**的后裔，属于初裔神"**梦嗣幻渊**"一脉的直系血统。

但她的存在本身便是一个悖论——幻渊是已知最早出现"血脉腐化"的初代神裔，他的意识能量变异为腐蚀性意识能量，引发了艾索斯历史上第一次血脉污染事件。按常理，纯系幻渊血脉的后裔应当是最危险的被污染者。

然而梦璃的血脉却呈现出一种极其罕见的状态——"**清醒幻境**"。

幻渊的腐化源于"无法区分感知现实与创造幻境的边界"，而梦璃天生具备一种被灵魂游走者称为"**梦之眼**"的特殊感知：她的左眼看到的永远是现实，右眼看到的永远是幻境。两者在她的意识中从不混淆、也从不重叠——她是幻渊血脉数千年来，第一个能同时站在"真"与"虚"两岸之间而不被吞噬的存在。

灵契编织者对她的评语是："她不是控制了幻境，而是她天生就活在两个世界的交界线上——对她而言，幻境不是工具，是另一只眼睛。"

### 2.2 身世背景

梦璃出生于人类王国"安索尔"的边境城镇"月牙镇"。她的母亲是镇上一名普通的织布匠人，父亲是一名游方学者——至少表面上如此。实际上她的父亲是幻渊血脉在人类社会中隐匿的最后一支"醒梦者"家族的末裔，一个只在血脉学者的密典中偶尔出现的古老名字。

"醒梦者"是幻渊一脉中极少数拒绝了腐化的血脉分支。他们的先祖在幻渊堕落之初，以一种近乎自残的方式**截断了与幻渊本体的意识连接**，代价是血脉力量被封印了九成以上。千年来，醒梦者一直以凡人的身份隐居在人类社会中，只在家族内部以口耳相传的方式保留着对血脉本质的理解。

梦璃三岁时，第一次展现出异常——她在睡梦中无意识地将整个月牙镇拉入了她的"梦境领域"。镇民们集体做了同一个梦：一个开满了琉璃色花朵的月光花园，蝴蝶振翅时散落的鳞粉如同星辰碎片。这个梦境美丽得不像梦，以至于第二天早晨醒来的镇民们都有一瞬间分不清现实与梦境——这正是幻渊腐化的前兆特征。

她的父亲意识到：被封印了千年的幻渊血脉，在梦璃体内完全苏醒了。而且其纯度远超醒梦者家族任何一代——她可能继承了比醒梦者先祖截断前更古老、更接近幻渊本体的血脉基因。

父亲在恐惧与希望之间做出了选择。他没有像祖训要求的那样"再次封印血脉"，而是花了十二年时间，将醒梦者家族口耳相传的所有血脉控制技术教给了梦璃——如何区分真实与幻境、如何在梦境中保持"我"的边界、如何用意志为幻渊的能量设定锚点使其不会失控。

十五岁时，梦璃已经能够自如地操控自己的梦境领域——让特定的人看到特定的幻象、在他人的梦中行走而不留下痕迹、甚至短暂地将现实中的物体"拉入"幻境维度使其暂时从物质界消失。

但这一切在她十六岁生日那天被打破。

一支"灵契编织者"的搜索队追踪到了"异常意识能量波动"的源头——月牙镇。他们来寻找的不是梦璃，而是任何可能的"幻渊腐化者"——因为近年来大陆各地出现了越来越多的"集体噩梦"事件，灵契编织者怀疑幻渊的腐化意识正在通过梦境渗透凡人世界。

搜索队抵达月牙镇时，梦璃的父亲主动现身，暴露了自己醒梦者的身份，以此引开追踪队伍。他对梦璃留下了最后一句话：

**"记住，你的右眼看到的一切都是假的——但假的并不意味着没有力量。去找暗影召唤师联盟，那里有能理解你的人。"**

父亲被灵契编织者带走后，梦璃独自离开月牙镇，一路向南，凭借幻境能力避开了所有追踪——她能让追踪者以为自己看到了她，而真正的她早已走向了另一个方向。

她在十八岁时抵达了召唤师联盟的驻地，并在入盟测试中展现出惊人的实力——测试官要求她战胜一名资深召唤师的三只召唤兽，而梦璃在战斗开始的瞬间让三只召唤兽同时陷入了"安宁之梦"——它们以为自己回到了最快乐的记忆中，主动停止了战斗。

测试官对她的评语是："这不是战斗力，这是降维打击。"

### 2.3 与其他角色的关系

| 角色 | 关系 |
|------|------|
| 暗影君主（主角） | 梦璃对暗影君主有着复杂的警惕与好奇。她的右眼能看到暗影君主身上潜藏的巨大力量轮廓，那个轮廓的形状让她想到父亲描述过的"初裔级能量体"。她是少数知道暗影君主可能不是普通人类的角色之一，但选择沉默观察。 |
| 绯夜 | 梦璃唯一主动保持距离的队友。绯夜的暗影之针能穿透大多数防御，而梦璃的幻境防御本质上是"意识层面的偏转"——两人的能力在理论上存在天然克制关系。绯夜的暗影不受幻觉影响，这让梦璃罕见地感到不安。但在战场上，两人反而配合绝佳——梦璃的幻境控制能将敌人困在原地，为绯夜创造完美的暗影之针射击窗口。 |
| 翎嫣 | 两人之间有一种奇妙的亲近感。翎嫣的自然能量有"抚慰意识"的副作用，在翎嫣身边时梦璃的右眼（幻境之眼）会短暂安宁下来，不再无休止地展示幻象。梦璃私下称翎嫣为"眼药水"——翎嫣对此困惑不已。 |
| 烬殇 | 梦璃对烬殇充满敬畏。烬殇的火焰不仅燃烧物质，也能灼烧意识——当梦璃试图对烬殇使用幻境时，她的幻象被烬殇的焰心意志直接焚毁。梦璃将烬殇列为"绝对不要惹的人·第一名"。 |
| 堕天使长 | 堕天使长的光系增伤标记对梦璃的幻境攻击同样有效——被标记的敌人在梦境维度中的防御会同步下降。两人在战术层面合作密切，但堕天使长对幻渊血脉有本能的不信任，梦璃对此心知肚明，也不刻意辩解。 |
| 命运织者 | 命运织者是梦璃在联盟中关系最亲近的角色。两人都拥有某种"超越常规感知"的能力——一个窥见命运，一个窥见幻境。深夜时两人经常在联盟塔楼的天台上聊天，互相描述自己"看到"的世界。命运织者曾说："你看到了所有可能发生的事，我看到了所有注定发生的事——我们之间，就是自由意志和宿命论的距离。" |

## 三、外形设计

### 3.1 整体风格

幻梦血脉的典型特征与"月下琉璃"的美学基调融合——梦璃的外貌给人的第一印象不是攻击性或威胁感，而是一种**不真实的美丽**，像是油画中走出的少女，又像是月光折射出的幻影。看到她的人会下意识产生"她是不是幻觉"的念头——这恰恰是幻渊血脉最本质的特征。

### 3.2 详细描述

| 特征 | 描述 |
|------|------|
| 身高 | 165cm，身形纤细修长，有一种轻盈到仿佛随时会飘起来的失重感。不是柔弱——是"轻"，像一片落在水面上的花瓣，存在感轻到恰好 |
| 肤色 | 瓷白通透，带极淡的月光蓝调，在阴影处能隐约看到皮肤下浅淡的琉璃色光脉流动——那是幻渊血脉的能量经络。日光下她的皮肤近乎完美无瑕，如同最上等的和田玉，温润、细腻、微微透光 |
| 发色 | **月白渐变琉璃紫**。发根是清冷的银白色，越向发尾越渐变为通透的琉璃紫——那种紫不是浓郁的皇室紫，而是透光的、像水晶一样可以看穿的淡紫。头发长度至腰际偏下，自然微卷，如同月光下的溪流。发间偶尔闪过细碎的光点，像是被风吹散的梦境碎片 |
| 瞳孔 | **异色双瞳**——这是幻渊血脉最显著的外在标志。**左眼为澄澈的琥珀金色**（现实之眼），清亮通透，像是封存了一枚微缩太阳的宝石。**右眼为流转的琉璃紫色**（幻境之眼），瞳孔中有微小的光点缓缓流动，像是一个微缩的星空在旋转。当她动用幻境能力时，右眼的紫色会变深，光点流动加速，偶尔会有极细的光丝从眼角逸出 |
| 睫毛 | 浓密纤长，自然翘曲，在眨眼时投下的阴影都是淡紫色的——这不是化妆效果，是幻渊血脉的能量渗透到了她身体的每一个细节 |
| 面容 | 鹅蛋脸型，轮廓柔和但五官精致。高而挺秀的鼻梁、微微上挑的眼尾（幻渊一脉的共同特征）、嘴唇是自然的樱粉色，微微含笑时有一种"仿佛什么都知道"的从容——像极了世界观中对薇嫣面容的描述，只是更年轻、更柔和 |
| 耳部 | 左耳佩戴一枚**月牙形琉璃耳坠**，内部封存着一缕流动的紫光——这是父亲离开时留给她的唯一遗物，内含醒梦者家族最后一代族长的一丝意识锚点。战斗中梦璃偶尔会触碰耳坠，仿佛在与它交流 |
| 手部 | 手指修长白皙，指尖有若隐若现的琉璃色光纹，施法时光纹延伸至手背。右手无名指上戴着一枚极简的银色指环——那是醒梦者家族的信物，戒面上刻着一个微小的、只有用意识能量才能看到的符号：一只闭合的眼睛 |
| 服饰 | 上身为合身的月白色短款立领上衣，领口和袖口绣有极淡的琉璃紫色梦纹图案——那是醒梦者家族代代相传的意识稳定纹路，既是装饰也是功能性符文。下身是渐变色长裙，从腰部的银白渐变到裙摆的透明琉璃紫，裙摆的材质在移动时会泛起微光，像是踩碎了一地月光。外搭一件轻薄的紫灰色披风，兜帽内侧绘有星图——那是幻渊一脉用来在梦境中定位"现实锚点"的导航图 |
| 足部 | 白色绑带式轻靴，靴面上有银色梦纹，脚步落地时会留下一瞬间的淡紫色光晕，像是踩在了幻境与现实的交界面上 |

### 3.3 战斗形态

当梦璃进入全力状态：

- 右眼的琉璃紫色完全绽放，紫光从眼角溢出，在脸侧形成流动的光纹
- 发色的琉璃紫渐变范围扩大，从发尾蔓延至发根，整头长发变为通透的星空紫色，发间的光点变为清晰可见的微型星辰
- 身后凝聚出一个模糊的巨大**琉璃色瞳孔幻象**——那是幻渊一脉的血脉幻象，一只看向所有人的眼睛。这个幻象的出现意味着梦璃将战场范围内所有生物的意识都纳入了她的"幻境视野"
- 裙摆和披风的下端开始溶解为透明的紫色光粒，像是她的存在正在"半步踏入幻境维度"

### 3.4 日常状态

脱离战斗时的梦璃看起来像一个安静的、略带迷糊的年轻女性。右眼的紫色流转会变得极其缓慢，像是一面安静的湖。她喜欢靠在窗边或树下半梦半醒地休憩——这不是懒惰，而是她在"整理"右眼无时无刻都在接收的幻境信息。

她的衣着在日常会更简朴——简单的白色连衣裙配月牙耳坠，看起来像是一个出身良好的普通少女。只有那双异色瞳孔暴露了她的不凡。

## 四、性格设计

### 4.1 核心性格

**安静而深邃的观察者。**

梦璃给人的第一印象是"温柔"和"安静"——她说话轻声细语，从不主动挑起争执，对谁都带着一丝若有若无的微笑。但与她相处久了的人会发现，她的"温柔"下面藏着一种让人背脊发凉的**洞察力**——她似乎总是知道别人在想什么。

这不是因为她读了你的心——而是因为她的右眼在无意识间不断展示着"你的幻象投影"——你潜意识中的情绪、欲望、恐惧，都会以幻象的形式映射在她的右眼视野中。她已经习惯了在对话时同时"看到"对方的表面和内在。

### 4.2 行为特征

| 特征 | 表现 |
|------|------|
| 说话习惯 | 语速偏慢，措辞精确，偶尔会在句中停顿——因为她在确认自己说的是"现实中该说的话"而不是"幻境中想到的话"。有时会突然说出看似不相关的话，过几天才被证实是"预知"——她的右眼偶尔会捕捉到未来的碎片 |
| 战斗态度 | 极其冷静，甚至被形容为"优雅"。她不喜欢暴力——用她的话说，"让他们在最美的梦中安睡，不是比打碎他们更好吗？"但当面对真正邪恶的敌人时，她会展现出"噩梦"的一面 |
| 个人爱好 | 收集各地的安眠花草，在自己的房间里种了一整面墙的月光藤。喜欢记录自己（以及偶尔窥见的他人的）梦境，已经写满了三本"梦录" |
| 弱点 | 对"分不清现实和幻境"有深层恐惧——这是对幻渊腐化命运的本能畏惧。如果有人刻意混淆她对现实的感知，她会罕见地惊慌 |
| 与人相处 | 表面上与所有人保持友善而适当的距离。真正亲近的人很少——因为她的右眼看到的"内在"太多了，大多数人的内心并不适合直视。她最放松的时刻是在翎嫣身边——自然能量能暂时平息右眼的幻象 |

### 4.3 经典台词

- （登场）"嘘……不要突然醒来，梦碎的声音很响的。"
- （技能释放·安宁之梦）"睡吧，这是我能给你的最温柔的结局。"
- （技能释放·万象幻境）"看好了——这是你自己的恐惧。"
- （受击）"嗯……是真的呢。"
- （击杀boss）"你做了一个很长的梦……不过，结束了。"

## 五、能力体系

### 5.1 核心机制：幻梦侵蚀（Dream Erosion）

梦璃的攻击不是传统的物理打击或法术轰击——她将意识涟漪注入目标，使其逐渐沉入幻境。在代码层面，这体现为一套**叠层控制 + 范围增益**的双线机制：

- **幻梦印记**（被动·OnHit）：普攻对命中目标叠加"幻梦印记"，满层后触发"沉梦"（眩晕），同时对周围敌人造成意识冲击
- **梦境共鸣**（被动·UpdateFrame）：范围光环持续为友方英雄提供攻速和暴击率加成，类似翎嫣的自然之力但采用直接增益而非渐近曲线
- **万象沉梦**（主动·TriggerActive）：全屏幻境爆发，造成高倍率伤害并眩晕，对已有幻梦印记的敌人额外加成

**与 Boss 的交互**（遵循 BOSS_BALANCE 规则）：
- 眩晕持续时间 ×0.50（stunDurationMult）
- 减速效率 ×0.50（slowEfficiency）
- Boss 免疫冰冻（freezeImmune），但幻梦眩晕走 stun 通道，不受冰冻免疫影响

### 5.2 战斗能力描述

从外部观察者角度，被梦璃攻击的敌人会出现短暂的迷离停顿——幻梦印记在目标头顶显示为琉璃色涟漪标记。当印记满层时，目标陷入"沉梦"状态，完全停止行动并承受爆发伤害。

这种叠层控制机制使梦璃在长期战斗中价值递增：攻速越快、目标越集中，她的控制频率越高。

| spd | 10 | 所有 UR 统一值 |

**成长系数**（UR 统一）：`growthPct = 0.15`，公式：`allStats × (1 + (level-1) × 0.15)`

**星级缩放**（starScale=true 的技能自管缩放）：`0.10 + 0.90 × sqrt(star / 30)`

## 九、平衡分析

### 9.1 功率预算验证

UR 功率预算为 16×（SSR 的 2 倍）。梦璃的等效 DPS 贡献分析：

| 技能 | 等效 DPS 贡献 | 说明 |
|------|-------------|------|
| 幻梦印记（被动） | ~4× | 每4次攻击触发250%ATK爆发 + 120%ATK溅射，含1.5秒眩晕（等效于中断敌人输出） |
| 梦境共鸣（光环） | ~6× | 攻速+25%和暴击率+15%对全队的等效 DPS 提升，持续性强 |
| 万象沉梦（主动） | ~6× | 700%ATK全屏 + 层数加成最高1020%，16秒冷却 |
| **合计** | **~16×** | 符合 UR 功率预算 |

### 9.2 与同定位英雄的差异化

| 对比项 | 梦璃 | 凛冬君王 | 翎嫣 |
|--------|------|---------|------|
| 控制方式 | **叠层触发眩晕**（需4次攻击叠满） | 即时减速+满层增伤 | 无控制，纯增益 |
| 增益覆盖 | 攻速+暴击率（中等强度） | 受伤+95%（被动增伤） | 攻击力+攻速（极高上限） |
| 主动技能 | 全屏伤害+眩晕（复合） | 无独立主动（全局寒意触发） | 全队自然之力+鲜花环 |
| 增长曲线 | 线性（印记叠满即生效） | 线性（寒意叠满即生效） | 渐近（自然之力半饱和曲线） |

### 9.3 弱点设计（防止过强）

1. **叠层依赖**：需要连续4次攻击同一目标才能触发眩晕，面对大量快速小怪时叠层效率低（印记分散）
2. **印记衰减**：5秒持续时间，如果攻速被降低或目标切换频繁，印记容易自然衰减归零
3. **眩晕对Boss减半**（BOSS_BALANCE.stunDurationMult=0.50）：对 Boss 只有0.75秒眩晕，控制窗口有限
4. **自身无输出保障**：ATK=3500 处于 UR 最低档，没有绯夜的暴击加成或烬殇的处决机制，纯输出能力偏弱
5. **光环范围限制**：110范围的光环不如翎嫣（140范围），布阵位置要求更高
6. **主动技能消耗印记**：万象沉梦释放后清空所有印记，等于"重置"叠层进度，需要重新积累

## 十一、设计总结

梦璃的设计核心是"**虚实之间的美丽裁决者**"。她的外貌美丽而不真实，性格温柔而深邃，能力体系围绕"梦境"这一主题构建了一套独特的渐进式控制机制。

在世界观层面，她代表了幻渊血脉的"另一种可能"——堕落并非宿命，在"真"与"虚"之间保持清醒的能力本身就是对幻渊腐化命运的最好回答。

在游戏性层面，她填补了人类阵营缺乏UR英雄的空白，也为游戏增添了一个全新的"控制/辅助"型UR定位——区别于翎嫣的纯辅助和凛冬君王的即时控制，梦璃的渐进式梦境浓度机制鼓励玩家在长期战斗中充分利用她的价值。

> "每个人都会做梦。区别只在于——谁的梦，由谁来决定。"
> ——梦璃
