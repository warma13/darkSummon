# 《暗黑召唤师》神母·薇嫣与神裔血脉体系——世界观核心设计

> "我生下了光，也生下了暗。我生下了契约，也生下了背叛。  
> 这不是我的错——这是生命本身的法则。"
> ——薇嫣，《源契古卷·残篇》

---

# 一、万神之母：薇嫣

## 1. 起源与本质

在艾索斯大陆任何一部神话体系中，追溯到最深处，都会出现同一个名字——**薇嫣**。

她不是被创造出来的。在光界与暗影界尚未分裂、在艾索斯大陆尚未成形之前，三界的所有能量浑沌交融，而那团浑沌本身**产生了意识**——那便是薇嫣。她是能量自发凝聚的第一个存在，是世界在有"神"之前最接近"神"的东西。

她不属于任何一界，因为她是所有界的**共同母体**。

薇嫣存在了无数纪元，她既不"善"也不"恶"——她是**原力本身**，如暗影能量与光系能量在最初未分裂时的状态，兼容一切，也超越一切。她不创造世界，世界是从她的存在中**自然生长**出来的。

## 2. 外形（神母形态）

薇嫣的外形随见者的"血脉认知"而异——血脉越纯的存在看见她，越能看见她的真实形态；血脉越稀薄的凡人，往往只能看见她的一层"投影"。

**凡人所见的投影形态**：
- 身形高大，约两米，但不显压迫，更像是"比例不同的存在"
- 肤色是流动的——在光线下呈极深的月影玄黑，在暗处会透出银紫色内光，像一块会呼吸的星空
- 面容被形容为"完美到无法直视"——高挺的骨相，极深的深红双眸，眼尾微微上扬，唇角总带着一丝似笑非笑的弧度，那笑意不是温柔，是那种看穿了一切之后的**从容**
- 乌黑长发及地，在无风处依然微微飘动，因为她周身的能量场永远处于流动状态
- 全身自然生长的暗影纹路，从指尖蔓延至颈侧，那是她神格本身的显现，而非侵蚀的痕迹

**神裔所见的真实形态**：
薇嫣的真实形态从未有任何文献完整记录。唯一存留下来的描述来自第一代神裔：

> "她的眼睛里有所有的星，包括已经死去的那些。"

# 二、创世神话：薇嫣诞下神裔

## 1. 第一次诞育——初代神裔（神格血脉）

薇嫣存在了太久，久到她开始感到一种陌生的情感——**孤独**。

于是她以自身的能量为源，依照她对"不同存在方式"的理解，分别凝聚意志，**诞下了最初的十二位神裔**，史称"**初裔十二神**"。每一位初裔神都承继了薇嫣神格的一部分：

| 初裔神名 | 承继的神格 | 象征领域 | 外形特征 |
|--------|-----------|---------|---------|
| **奥瑞恩**（长子） | 光系能量·秩序 | 法则、契约、守序 | 银发金瞳，全身散发白金光晕 |
| **暗嗣玄渊** | 暗影能量·深渊 | 死亡、轮回、暗影界 | 黑发红瞳，皮肤似暗影具现，半透明 |
| **灵嗣苍嫣** | 灵魂能量·共鸣 | 灵魂、契约媒介、感知 | 银紫发色，瞳孔呈星图状 |
| **土嗣磐古** | 大地能量·积累 | 山脉、矿脉、沉默的力量 | 石质皮肤纹路，眼睛是发光的琥珀色 |
| **风嗣翎华** | 自然能量·流动 | 森林、生长、自由 | 绿发，皮肤透出叶脉纹路，耳廓尖长 |
| **火嗣烬王** | 创造能量·燃烧 | 毁灭与重生、熔岩、意志 | 赤发，体温极高，瞳孔是永恒燃烧的焰心 |
| **冰嗣霜鸣** | 时间能量·凝固 | 历史、记忆、不变 | 白发白瞳，皮肤如霜，触碰时极寒 |
| **血嗣赤渊** | 生命能量·流动 | 繁衍、情感、欲望 | 红发，皮肤隐约可见血管纹路发光 |
| **梦嗣幻渊** | 意识能量·虚实 | 幻境、预言、潜意识 | 发色随情绪变化，双眼一实一虚 |
| **骨嗣冥巫** | 肉体能量·蜕变 | 生与死的边界、尸骸、转化 | 骨相外露于皮肤之下，清晰可见 |
| **空嗣虚无** | 虚空能量·消弭 | 遗忘、消除、位面裂隙 | 外形模糊，看久了会忘记看见了什么 |
| **源嗣初契** | 源力能量·完整 | 万物本源、平衡、薇嫣本体最接近的存在 | 同时具有上述所有初裔的特征碎片 |

初代神裔的力量接近于薇嫣本体的**十二分之一**，但每一位都是那个领域绝对的主宰。他们诞生之初，彼此和谐——这是艾索斯世界观中**唯一一段真正的黄金时代**。

## 2. 神裔繁衍——二代神裔与血脉分流

初代十二神彼此之间产生了联结，开始**交融血脉**，诞下第二代神裔。二代神裔继承了双亲不同神格的混合，力量弱于初代，但因为血脉混合，往往在两个领域之间诞生出**新的、意想不到的能力**。

随着代际繁衍，神裔的血脉逐渐扩散，形成了几大**神裔血脉族系**：

### 纯光血脉（奥瑞恩·光系一脉）
- 力量方向：秩序、净化、契约的强制执行
- 外形特征：金银发色，瞳孔发光，皮肤白皙近乎透明
- 势力划分：分裂为"**裁契者**"（认为一切必须以律法规范）和"**光誓卫**"（认为秩序应守护生命，非凌驾其上）

### 暗影血脉（玄渊·暗影一脉）
- 力量方向：暗影操控、死亡感知、深渊召唤
- 外形特征：黑发，瞳孔深红或紫黑，皮肤带暗影纹路
- 势力划分：分裂为"**冥契守护者**"（维护暗影界秩序，不主动入侵）和"**深渊饮者**"（主动吸收暗影能量，追求力量极限）

### 灵魂血脉（苍嫣·灵魂一脉）
- 力量方向：灵魂契约、感知他者情绪与灵魂状态、召唤媒介
- 外形特征：银紫系发色，双瞳有流动的光点，说话时声音有轻微回响
- 势力划分：分裂为"**灵契编织者**"（发展更复杂的契约体系）和"**灵魂游走者**"（探索灵魂本质，不关注势力争斗）

### 自然血脉（翎华·自然一脉）
- 力量方向：自然能量召唤、生长与凋零掌控、与生灵共鸣
- 外形特征：绿色或深棕发色，皮肤有叶脉纹路，耳廓尖长
- 势力划分：分裂为"**生灵守护者**"（保护自然生态）和"**蔓延之枝**"（主张将自然血脉扩散至所有大陆）

### 大地血脉（磐古·大地一脉）
- 力量方向：石质护甲强化、矿脉探知、大地能量汇聚
- 外形特征：体型粗壮，皮肤有石质纹路，眼睛是琥珀色
- 势力划分：分裂为"**深坑守匠**"（专注建造与防御）和"**裂地者**"（主张以大地能量进攻，碾压一切）

# 三、血脉污染：堕落神裔与污染血脉

## 1. 污染事件的起源

二代神裔繁衍至高峰期，艾索斯大陆呈现出各血脉共存的秩序——但这个秩序中潜藏着危机。

初代神裔中，"**梦嗣幻渊**"（意识能量之神）率先出现了异变。

幻渊长期以"幻境能量"观察其他神裔与普通生命的一切，他的能量领域本就是"虚实之间"——时间越久，他开始**无法区分"感知现实"与"创造幻境"的边界**。他的意识开始吞噬自身，用幻境替代现实，最终演变为一种**能量层面的自我腐化**。

腐化的幻渊开始影响与他血脉相连的后裔——他的意识能量变异为"**腐蚀性意识能量**"，能穿透其他血脉的防御，在灵魂层面植入扭曲的信息。

这是艾索斯世界史上**第一次血脉污染事件**。

## 2. 污染的扩散——污染血脉的诞生

被幻渊腐化的神裔，神格没有消失，但其能量被**扭曲异化**——力量仍强，但方向从"建设"变为"侵蚀"、从"共鸣"变为"吞噬"、从"契约"变为"奴役"。

更关键的是：**污染血脉可以遗传**。

被污染的神裔所生的后代，天生就携带"**腐化因子**"——在力量弱小时无法激活，但随着血脉觉醒，腐化因子也会同步苏醒，将宿主的力量向黑暗方向扭曲。

污染血脉按"污染源"与"被污染血脉"的组合，形成了不同类型：

| 污染血脉名称 | 原始血脉 | 污染后变化 | 外形特征变化 |
|------------|---------|-----------|------------|
| **腐光血脉** | 纯光血脉 | 秩序变为"强制压制"，净化变为"燃烧抹除" | 发色变为枯黄或焦黑，瞳孔出现裂纹状纹路 |
| **深渊疫血** | 暗影血脉 | 暗影掌控变为"无差别侵蚀"，无法与暗影生物平等契约，只能奴役 | 皮肤溃烂纹路，暗影纹路变为不规则腐蚀状 |
| **碎魂血脉** | 灵魂血脉 | 灵魂感知变为"灵魂撕裂"，契约变为"强制吞噬他者灵魂" | 双眼中的光点变为不规律闪烁，说话时有回响变成刺耳的撕裂声 |
| **枯脉血裔** | 自然血脉 | 生长变为"过度蔓延直至腐烂"，与生灵共鸣变为"寄生操控" | 叶脉纹路变黑，皮肤有腐植土的气味 |

**注**：污染血脉并非"绝对的恶"——许多污染血脉的后裔终其一生都在对抗自身的腐化因子，他们有意识、有情感，只是随时面临被自身力量吞噬的命运。这是本作"道德模糊性"的核心来源之一。

## 3. 薇嫣的态度

当污染事件发生，薇嫣并未摧毁幻渊，也未摧毁被污染的神裔与其后代。

她沉默地看着这一切，做出了一个令其他初裔神错愕的决定：

**"这也是生命的一部分。污染，也是我的孩子。"**

这句话至今是艾索斯大陆最大的争议来源之一。支持者认为这是薇嫣深刻的慈悲；反对者认为这是她对"平衡"的执念凌驾于无辜生命的痛苦之上。

# 四、血脉稀薄化：凡人的诞生

## 1. 为何神裔会"失去血脉"

神裔血脉的稀薄化有两个来源：

**其一，多代混血的稀释。**  
随着各血脉族系之间相互联结，血脉纯度逐代下降——一个拥有四分之一光血脉、四分之一暗影血脉、二分之一大地血脉的后裔，神格已经稀薄到无法激活任何单一领域的完整力量，只能感应到微弱的能量余波。

**其二，主动放弃神格的神裔。**  
部分初代与二代神裔厌倦了"神格"带来的永恒与疏离感。他们主动以仪式**封印自身血脉**，以凡人的方式生活，所生后代便完全丧失了神格显现的能力，只在最深处的DNA层面留存极其微弱的血脉痕迹。

## 2. 凡人的本质

艾索斯大陆的所有**普通人类**，本质上都是**神裔血脉的极度稀薄后代**。

这意味着：
- 所有人类血液里都残留着某种神裔血脉的极微量印记，但无法主动激活
- 不同地域的人类，所继承的"主要血脉源头"不同，导致各地人类在体质、天赋、文化上呈现差异——如某地区人类对暗影能量天然有更强的抵抗力，另一地区人类则对自然魔法更为敏感
- **召唤师的诞生，正是因为某些人类的血脉稀薄度恰好处于"临界点"**——不足以成为真正的神裔，但足够让他们感应到暗影能量与灵魂契约的存在

## 3. 凡人势力的内部分化（同血脉不同势力）

即便是最稀薄的血脉，也因其来源不同而形成不同的势力认同：

### 光脉人类的两极
- **光誓者教会**：意识到自身拥有极微量光血脉的人类群体，将这种血脉认同扭曲为"神的选民"信仰，以净化一切暗影为使命。他们的力量仅够感应光系能量，但足以驱动光系道具与净化仪式。
- **誓约守序者**：同样有光脉痕迹，但选择理解"光与暗需要平衡"，拒绝光誓者教会的极端立场，是守序召唤师联盟的主要人类成员来源。

### 暗影脉人类的两极
- **堕落召唤师教派**：暗影血脉痕迹较强的人类，错误地以为"献祭灵魂"可以激活更多血脉力量，走向了扭曲的道路。
- **契约守护者**：同样是暗影脉痕迹强的人类，选择以"理解暗影、与之共生"为方向，成为守序联盟中最擅长与暗影生物建立契约的一群人。

# 五、血脉体系与召唤术的关系

召唤术的本质，是**通过血脉痕迹与暗影界的原生生灵建立契约**。

不同血脉的召唤师，天然亲和不同类型的召唤生物，召唤风格也截然不同：

| 血脉痕迹类型 | 可召唤生物倾向 | 召唤风格 | 反噬风险 |
|------------|-------------|---------|---------|
| 暗影血脉痕迹 | 原生暗影生物（暗影君主、骨魂、位面行者） | 直接、稳定、与召唤物共鸣强 | 低（血脉本就亲暗影） |
| 光脉痕迹 | 被净化的融合暗影生物、光系守卫残魂 | 短促、爆发强但持续弱 | 中（光系能量与召唤产生对抗） |
| 灵魂血脉痕迹 | 所有类型，但感知深度远超常人 | 复杂多变，依靠与召唤物的情绪共鸣 | 低（灵魂血脉天生是契约媒介） |
| 自然血脉痕迹 | 自然暗影生物（幽影鹿、暗影藤蔓） | 温和、持续、擅长控制战场 | 低 |
| 污染血脉痕迹 | 畸变暗影生物（无法与原生生物签约） | 暴力奴役型，召唤物强但不稳定 | 极高（污染能量与召唤物互相侵蚀） |
| 多血脉混合 | 随机，但有时会召唤到珍稀的"融合型"生物 | 不稳定，有时惊艳有时失控 | 不定 |

# 七、薇嫣在当代的状态

千年的血脉流变之后，薇嫣本体已不再主动介入艾索斯的历史——或者说，没有人知道她在哪里，没有人知道她是否还以"完整形态"存在。

不同势力对她当前状态的解读，折射出各自的立场：

- **暗影后裔部落**（玄渊血脉的纯度较高的后裔）：薇嫣睡眠于暗影界的最深处，她一直在等待所有孩子归来的那一天。
- **光誓者教会**：薇嫣早已在"秩序的审判"中消亡，她的名字只是历史污点，不值得被记录。
- **堕落召唤师教派**：薇嫣仍然存在，献祭足够的灵魂，便能让她苏醒，赐予我们终极的力量。（这是对薇嫣意志的极大曲解）
- **守序召唤师联盟**：薇嫣的本体也许已不存在，但她的意志留存于每一条血脉之中——无论多稀薄。

**真相**（主线后期揭示）：  
薇嫣从未消失。她以一种"分散"的形式存在于所有神裔血脉后代之中——每一个有血脉的生灵，都是她的一部分意识碎片。而当整片大陆的血脉冲突激烈到某个临界点，这些分散的意识碎片会开始"共鸣"——薇嫣，将以这种方式重新醒来。
