# 无尽深渊 - 系统设计文档

> **定位**: 纯战力检验型无尽波次关卡，永远无法通关，玩家追求最高波次纪录
>
> **世界观锚点**: 深渊深处存在一道被神母薇嫣封印的「梦渊裂隙」——梦嗣幻渊堕落时撕开的意识裂缝。裂隙不断涌出被腐蚀意识驱动的深渊生物，越深层的生物受到的腐化越彻底，拥有越扭曲的能力。召唤师必须阻止它们溢出深渊，每一次挑战都是对裂隙边缘的一次镇压。

---

## 一、核心玩法概述

### 1.1 基本规则

| 项目 | 说明 |
|------|------|
| 入场条件 | 主线通关第 5 章 |
| 挑战次数 | 每日 1 次免费，每多 1 次消耗 1 张深渊门票 |
| 胜利条件 | 无（无尽模式，无法通关） |
| 失败条件 | 场上存活英雄数为 0，或场上敌人数量超过 30 持续 10 秒 |
| 核心目标 | 推进最高波次，刷新个人/服务器纪录 |
| 战斗模式 | 与主线相同的合并召唤战斗系统 |

### 1.2 波次结构

```
波次 1-∞，持续出怪，无间歇

每波包含：
- 普通波（大部分）: 小怪群，数量随波次递增
- BOSS 波（每 10 波）: 先出前哨小怪 → 暂停 2s → BOSS 登场 → 暂停 3s → 增援小怪
- 精英波（每 5 波，非 BOSS 波）: 普通小怪 + 1~2 只精英怪（带词缀）
```

**波次时间线示例**:
```
Wave 1-4:   纯小怪
Wave 5:     精英波（小怪 + 精英×1）
Wave 10:    BOSS 波（前哨→骨龙→增援）
Wave 11-14: 纯小怪
Wave 15:    精英波（小怪 + 精英×2）
Wave 20:    BOSS 波（前哨→暗影领主→增援）
...
```

## 二、怪物数值设计

### 2.1 小怪基础属性

小怪的角色类型沿用现有怪物主题系统（亡灵、熔岩、森林、冰霜、虚空），按波次循环切换主题。

| 属性 | 基础值 | 增长公式 | 说明 |
|------|--------|---------|------|
| **生命值** | 100 | `baseHP × (1 + 0.15 × wave) × (1.08 ^ wave)` | 线性 + 指数复合增长 |
| **防御力** | 10 | `baseDEF × (1 + 0.08 × wave) × (1.04 ^ wave)` | 增长慢于 HP |
| **移动速度** | 50 | `baseSpeed × (1 + min(wave × 0.003, 0.4))` | 上限 +40% |
| **数量/波** | 4 | `min(4 + floor(wave × 0.25), 15)` | 上限 15 只/波 |

**HP 增长参考表**:

| 波次 | HP 倍率 | 实际 HP（基础 100） |
|------|---------|-------------------|
| 1 | 1.24× | 124 |
| 10 | 4.86× | 486 |
| 25 | 27.2× | 2,720 |
| 50 | 286× | 28,600 |
| 100 | 29,760× | 2,976,000 |
| 200 | 3.16 亿× | 316 亿 |

### 2.2 小怪类型轮换

每 20 波为一个主题循环，每个主题持续 4 波（BOSS 波使用当前主题）：

```
Wave  1-4:   亡灵墓地（灰绿色）
Wave  5-8:   熔岩地狱（暗红橙）
Wave  9-12:  幽暗森林（暗绿紫）
Wave 13-16:  冰霜冻土（冰蓝色）
Wave 17-20:  虚空深渊（深紫色）
Wave 21-24:  亡灵墓地（循环）
...
```

每个主题的小怪角色池随波次解锁（沿用现有怪物角色系统）：

| 波次 | 新增角色 |
|------|---------|
| 1+ | 杂兵(Minion)、步兵(Infantry) |
| 15+ | 坦克(Tank)（带主题被动） |
| 30+ | 刺客(Assassin) |
| 50+ | 闪避者(Dodger)（30% 闪避） |
| 70+ | 辅助者(Support)（主题光环） |
| 90+ | 分裂者(Splitter)（死亡分裂为 2 只杂兵） |
| 110+ | 闪现者(Blinker)（8% 概率闪现） |
| 130+ | 特殊者(Special)（主题独有能力） |

## 三、BOSS 设计

### 3.1 BOSS 出场规则

- 每 **10 波** 出现一个 BOSS
- 5 个 BOSS 按固定顺序循环：骨龙 → 暗影领主 → 深渊巨像 → 虚空织者 → 末日使者
- 每完成一轮循环（50 波），BOSS 进入下一个**阶段**，获得更强的数值和额外技能

### 3.2 BOSS 基础属性

| BOSS | 世界观身份 | 基础 HP | 基础防御 | 速度 | 体型 |
|------|-----------|---------|---------|------|------|
| **骨龙·亡骸吞噬者** | 亡灵墓地守卫，被梦渊腐蚀的远古龙骸 | 2,000 | 50 | 25 | 大 |
| **暗影领主·玄渊使徒** | 玄渊血脉的堕落使徒，操纵暗影军团 | 2,500 | 40 | 28 | 大 |
| **深渊巨像·磐古残躯** | 磐古血脉的腐化巨人，纯粹的肉体堡垒 | 5,000 | 80 | 18 | 巨大 |
| **虚空织者·幻渊碎念** | 梦嗣幻渊的意识碎片，在虚实间穿梭 | 1,500 | 30 | 40 | 中 |
| **末日使者·裂隙先驱** | 裂隙深处涌出的终极先兆体 | 3,500 | 60 | 30 | 大 |

**BOSS HP 增长**:

## 八、数值平衡备注

### 8.1 设计目标

| 玩家阶段 | 预期推进波次 |
|---------|------------|
| 新手（1 周内） | 15-25 波 |
| 成长期（1 个月） | 40-60 波 |
| 中期玩家（3 个月） | 80-120 波 |
| 后期玩家（6 个月+） | 150-200 波 |
| 极限玩家（全满配） | 250+ 波 |

### 8.2 关键平衡点

1. **HP 增长曲线** `(1 + 0.15w) × 1.08^w` 确保前 50 波可感知增长、50 波后急剧攀升
2. **防御增长慢于 HP**，让穿甲属性在后期不至于完全无效
3. **BOSS 阶段倍率** ×3/×8/×20 制造明显的"墙"感，玩家需要战力突破才能跨越
4. **环境负面效果**逐层叠加，50 波一个质变点，给玩家持续的压迫感
5. **毁灭技能（末日使者/第六层+）** 是终极压力源，迫使玩家在高波次维持大量高星英雄
6. **降星诅咒**临时性（8s），给予反应窗口但在高压下极为致命

### 8.3 与现有系统的数值对齐

- 小怪 HP 增长公式与主线怪物增长保持同系（主线用 `0.12w × 1.07^w`，深渊略快）
- BOSS HP 增长与世界 BOSS 同量级，但世界 BOSS 是固定 HP，深渊 BOSS 随波次成长
- 奖励产出不超过每日广告收益的 50%，定位为补充收入而非主要来源

## 九、技术实现要点

### 9.1 新增数据模块

- `AbyssData.lua` — 存档数据（最高波次、当日次数、里程碑领取状态）
- `AbyssConfig.lua` — 配置常量（BOSS 定义、词缀池、奖励表、增长公式）
- `AbyssBattle.lua` — 战斗逻辑（波次调度、BOSS 技能执行、环境效果）
