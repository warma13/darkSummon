## 1. 系统概览

装备系统为每个**随从英雄**（不含主角）提供 4 件装备，通过消耗噬魂石升级装备等级、突破品质段，获得额外属性加成。

```
英雄 ─┬─ 武器（攻击）
      ├─ 铠甲（血量）
      ├─ 头盔（防御）
      └─ 坐骑（血量）

升级: 噬魂石 → 等级+1 → 属性增长
突破: 噬魂石 → 品质↑ → 解锁更高等级上限 + 属性倍率提升
套装: 4件同品质 → 百分比属性加成
```

### 核心规则

1. **主角不参与装备系统**，仅随从英雄可装备
2. **装备等级不能超过英雄等级**，升英雄才能继续升装备
3. **升级和突破均消耗噬魂石**（与英雄进阶共用资源）

## 2. 装备部位

每个英雄固定 4 个装备位，初始均为 1 级腐骨品质。

| 部位 | Emoji | 主属性 | 说明 |
|------|-------|--------|------|
| 武器 | ⚔ | 攻击 (atk) | 提升英雄攻击力 |
| 铠甲 | 🛡 | 血量 (hp) | 提升英雄生命值 |
| 头盔 | ⛑ | 防御 (def) | 提升英雄防御力 |
| 坐骑 | 🐴 | 血量 (hp) | 提升英雄生命值 |

5 个品质段，等级到达上限后需突破进入下一品质。

| 蓝 | 冥铁 | 蓝 | 201~1000 | 50 | 冥铁阔剑 | 冥铁锁甲 | 冥铁角盔 | 幽蓝噩马 |
| 紫 | 噬魂 | 紫 | 1001~2000 | 200 | 噬魂弯刀 | 噬魂胸甲 | 噬魂冠冕 | 暗影梦魇 |

## 4. 升级与突破

### 升级

消耗噬魂石，每次升 1 级。支持一键升级（升满当前品质段或英雄等级上限）。

**升级费用（噬魂石/级）：**

| 等级区间 | 费用/级 |
|----------|---------|
| 1~200 | 1 |
| 201~1000 | 2 |
| 1001~2000 | 4 |
| 2001~3000 | 8 |
| 3001~4000 | 15 |

**等级上限约束：**

```
装备可升级最高等级 = min(当前品质段上限, 英雄当前等级)
```

- 英雄 Lv.50 → 装备最高只能升到 50 级
- 装备到达品质段上限（如绿色200级）→ 需要突破才能继续

### 突破

等级到达品质段上限后，消耗噬魂石突破到下一品质。突破后解锁更高等级上限，属性倍率提升。

```
绿色(200级满) ──突破50石──→ 蓝色(可升到1000)
蓝色(1000级满) ──突破200石──→ 紫色(可升到2000)
紫色(2000级满) ──突破500石──→ 橙色(可升到3000)
橙色(3000级满) ──突破1000石──→ 红色(可升到4000)
```

## 5. 套装系统

4 件装备中**品质最低的那件**决定当前套装等级。当 4 件装备都达到该品质或更高时，激活套装加成。

### 套装加成

| 套装 | 激活条件 | 加成效果 |
|------|----------|----------|
| 腐骨套装 | 4件≥绿 | 血量 +5% |
| 冥铁套装 | 4件≥蓝 | 血量 +10% |
| 噬魂套装 | 4件≥紫 | 血量 +15%, 攻击 +5% |
| 渊狱套装 | 4件≥橙 | 血量 +20%, 攻击 +10% |
| 灭世套装 | 4件≥红 | 血量 +30%, 攻击 +15%, 防御 +10% |

### 判定规则

```
4件品质: [蓝, 蓝, 蓝, 绿]
  → 最低品质 = 绿
  → 激活"腐骨套装" (4件≥绿 ✓)
  → 不激活"冥铁套装" (4件≥蓝 ✗，坐骑还是绿)
```

## 6. 属性计算

### 单件属性

```
属性加成 = 基础值 × 等级 × 品质倍率
```

| 属性 | 基础值 |
|------|--------|
| 攻击 (atk) | 12 |
| 血量 (hp) | 50 |
| 防御 (def) | 13 |

| 品质 | 倍率 |
|------|------|
| 绿(腐骨) | ×1.0 |
| 蓝(冥铁) | ×1.5 |
| 紫(噬魂) | ×2.0 |
| 橙(渊狱) | ×3.0 |
| 红(灭世) | ×5.0 |

**示例：** 蓝色武器 Lv.500 → 攻击加成 = 12 × 500 × 1.5 = **9000**

### 总加成

英雄装备总属性 = 4件装备的固定值加成之和 + 套装百分比加成

```lua
EquipData.GetTotalBonus(heroId)
-- 返回: { atk=N, hp=N, def=N, atk_pct=0.15, hp_pct=0.30, def_pct=0.10 }
```

## 7. 数据存储

装备数据挂载在 `HeroData.equipData` 上，随存档保存/加载。按英雄ID索引。

```lua
HeroData.equipData = {
    ["skeleton"] = {                -- 英雄ID
        weapon = { level = 150, tierIdx = 1 },   -- 绿色150级
        armor  = { level = 200, tierIdx = 2 },   -- 蓝色200级（已突破）
        helmet = { level = 100, tierIdx = 1 },   -- 绿色100级
        mount  = { level = 180, tierIdx = 1 },   -- 绿色180级
    },
    ["demon"] = {
        weapon = { level = 1, tierIdx = 1 },
        armor  = { level = 1, tierIdx = 1 },
        helmet = { level = 1, tierIdx = 1 },
        mount  = { level = 1, tierIdx = 1 },
    },
}
```

### 懒初始化

首次访问某英雄装备时自动初始化为 4 件 1 级绿色装备，无需预先创建。

## 8. UI 界面

### 页面结构

```
┌──────────────────────────────┐
│  装备               ⚗噬魂石 0 │
├──────────────────────────────┤
│  [骷髅] [恶魔] ...  英雄选择  │
├──────────────────────────────┤
│  ┌─────────────────────────┐ │
│  │ ⚔ 腐骨短刃        [升级] │ │
│  │  攻击 +12          ⚗1   │ │
│  │  腐骨套装 4/4            │ │
│  ├─────────────────────────┤ │
│  │ 🛡 腐骨皮甲        [升级] │ │
│  │  血量 +50          ⚗1   │ │
│  │  腐骨套装 4/4            │ │
│  ├─────────────────────────┤ │
│  │ ⛑ 腐骨面罩        [升级] │ │
│  │  防御 +13          ⚗1   │ │
│  │  腐骨套装 4/4            │ │
│  ├─────────────────────────┤ │
│  │ 🐴 亡骸幽马        [升级] │ │
│  │  血量 +50          ⚗1   │ │
│  │  腐骨套装 4/4            │ │
│  └─────────────────────────┘ │
├──────────────────────────────┤
│  [腐骨套装]  血量+5%         │
├──────────────────────────────┤
│         [ 一键升级 ]          │
└──────────────────────────────┘
```

### 按钮状态

| 状态 | 按钮显示 | 样式 |
|------|----------|------|
| 可升级 | "升级" + 费用 | 高亮(primary) |
| 到达品质上限 | "突破" + 突破费用 | 描边(outline) |
| 到达英雄等级上限 | "升级" | 置灰(ghost)，不显示费用 |
| 4000满级 | "满级" | 置灰(ghost) |

### 交互说明

- 顶部英雄选择栏横向滚动，仅显示已上阵随从英雄（不含主角）
- 选中英雄后刷新 4 件装备卡片
- 点击升级：单次升 1 级，消耗噬魂石
- 点击突破：消耗噬魂石进入下一品质段
- 一键升级：所有 4 件装备升到当前品质段上限或英雄等级上限
