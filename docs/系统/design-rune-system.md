# 符文系统设计文档

> **暗影召唤师 - 新装备系统：深渊符文**

---

## 一、设计目标

| 目标 | 说明 |
|------|------|
| **独立渠道产出** | 新增"深渊裂隙"副本，专门掉落符文 |
| **随机词条** | 每个符文携带 2-4 条随机词条，数值随机浮动 |
| **洗练系统** | 消耗特殊材料"裂隙之尘"重随词条，保留锁定词条 |
| **构建多样性** | 不同词条搭配催生不同英雄流派，增加游戏深度 |
| **长期追求** | 追求"完美词条"提供持续的刷取动力 |

## 二、符文概览

### 2.1 符文栏位

每个英雄拥有 **3 个符文槽位**（独立于现有4件装备），按副本进度解锁：

| 槽位 | 名称 | 解锁条件 |
|------|------|---------|
| 槽1 | 源力核心 | 通关主线第 5 章（第100波） |
| 槽2 | 意志铭刻 | 通关主线第 10 章（第200波） |
| 槽3 | 深渊之印 | 通关主线第 20 章（第400波） |

### 2.2 符文品质

| 品质 | 颜色 | 初始词条数 | 最大词条数 | 掉落权重 |
|------|------|-----------|-----------|---------|
| 白色（普通） | `{180,180,180}` | 1 | 2 | 40% |
| 绿色（精良） | `{80,200,80}` | 2 | 3 | 30% |
| 蓝色（稀有） | `{80,140,255}` | 2 | 3 | 18% |
| 紫色（史诗） | `{180,80,255}` | 3 | 4 | 8% |
| 橙色（传说） | `{255,160,40}` | 3 | 4 | 3% |
| 红色（神话） | `{255,50,50}` | 4 | 4 | 1% |

### 2.3 符文系列（套装效果）

每个符文属于一个**系列**，装备同系列符文可激活套装效果：

| 系列 | 标签色 | 2件套 | 3件套 |
|------|--------|------|------|
| 🔥 **烈焰** | `{255,100,40}` | 攻击力 +8% | 攻击附带灼烧DOT（每秒2%攻击力，持续3秒） |
| 🧊 **寒霜** | `{100,180,255}` | 攻速 +12% | 攻击有15%概率冻结敌人1秒 |
| ⚡ **雷霆** | `{200,180,60}` | 暴击率 +10% | 暴击时对周围敌人造成30%溅射伤害 |
| 🌑 **暗影** | `{160,80,200}` | 伤害加成 +10% | 击杀时15%概率双倍掉落（暗魂） |
| 💀 **亡灵** | `{140,180,140}` | 穿甲 +8% | 每次攻击叠加诅咒，5层后触发额外真实伤害 |
| 🛡️ **铁壁** | `{180,180,200}` | 所有英雄攻击力 +3% | 受到致命伤害时，10%概率免疫（每30秒最多触发1次） |

## 三、词条系统

### 3.1 词条池

词条分为**基础属性**和**特殊效果**两大类：

#### 基础属性词条（权重60%）

| 词条 | 数值范围 | 单位 | 说明 |
|------|---------|------|------|
| 攻击力 | 3% ~ 15% | % | 乘算最终攻击力 |
| 攻击速度 | 3% ~ 12% | % | 降低攻击间隔 |
| 暴击率 | 2% ~ 10% | % | 提升暴击概率 |
| 暴击伤害 | 5% ~ 25% | % | 提升暴击倍率 |
| 穿甲 | 2% ~ 10% | % | 忽略敌人防御 |
| 伤害加成 | 3% ~ 12% | % | 独立乘区 |
| 攻击范围 | 5 ~ 20 | px | 增加索敌距离 |

#### 特殊效果词条（权重40%）

| 词条 | 数值范围 | 说明 |
|------|---------|------|
| 连锁概率 | 5% ~ 20% | 攻击后有概率对附近第二个敌人造成50%伤害 |
| 减速强化 | 5% ~ 15% | 增加减速效果的减速率 |
| DOT强化 | 10% ~ 30% | 增加持续伤害效果 |
| 技能冷却缩减 | 3% ~ 10% | 降低主动技能冷却时间 |
| 击杀回复 | 1% ~ 5% | 击杀后降低当前攻击冷却的百分比 |
| 易伤标记 | 3% ~ 8% | 攻击有概率使目标受到额外伤害 |
| 元素精通 | 5% ~ 15% | 增强英雄元素克制效果 |
| 幸运掉落 | 2% ~ 8% | 增加怪物掉落锻魂铁的概率 |

### 3.2 词条数值生成规则

每条词条的最终数值 = `基础范围 × 品质系数 × (0.8 + random() × 0.4)`

**品质系数**：

| 品质 | 系数 | 说明 |
|------|------|------|
| 白色 | 0.6 | 只能Roll到低值 |
| 绿色 | 0.75 | |
| 蓝色 | 0.85 | |
| 紫色 | 1.0 | 标准值 |
| 橙色 | 1.15 | 可能超过基础范围上限 |
| 红色 | 1.35 | 词条闪光，数值极高 |

### 3.3 词条规则

- 同一符文**不能出现完全相同的词条**
- 基础属性 + 特殊效果各自独立Roll
- 每个词条有独立品质（不跟符文品质绑定），影响数值颜色显示

## 四、深渊裂隙副本

### 4.1 基本信息

| 属性 | 值 |
|------|---|
| 名称 | 深渊裂隙 |
| 图标 | 紫色漩涡 |
| 每日次数 | 3次（1次免费 + 2次观看广告） |
| 波数 | 15波 |
| 每波敌人 | 25 |
| 解锁条件 | 通关主线第 5 章 |

### 4.2 难度等级

每次进入可选择难度，影响符文品质和词条数量：

| 难度 | 敌人等级 | 符文品质加成 | 额外材料 |
|------|---------|------------|---------|
| 普通 | 关卡×0.8 | 无 | 裂隙之尘 ×3~5 |
| 困难 | 关卡×1.2 | 紫色以上概率 ×1.5 | 裂隙之尘 ×6~10 |
| 噩梦 | 关卡×1.8 | 橙色以上概率 ×2.0 | 裂隙之尘 ×10~15 |

> "关卡"指玩家当前主线最高通关关卡。

### 4.3 掉落规则

| 波次 | 掉落 |
|------|------|
| 波 1-4 | 裂隙之尘 ×1~2 |
| 波 5（精英波） | 裂隙之尘 ×3 + 10%概率掉落符文 + 10%概率掉落符文封印 ×1 |
| 波 6-9 | 裂隙之尘 ×1~2 |
| 波 10（精英波） | 裂隙之尘 ×5 + 20%概率掉落符文 + 15%概率掉落符文封印 ×1 |
| 波 11-14 | 裂隙之尘 ×2~3 |
| 波 15（BOSS波） | **必定掉落1个符文** + 裂隙之尘 ×8~12 + 20%概率掉落符文封印 ×1~2 |

**每次通关保底**：1个符文（BOSS波） + 约 30~50 裂隙之尘 + 约 0~2 符文封印

### 4.4 BOSS 设计

深渊裂隙 BOSS 随机从以下 BOSS 池中选取（每日轮换）：

| BOSS | 特殊机制 | 推荐应对 |
|------|---------|---------|
| 裂隙守卫 | 每20秒生成护盾，需集火破盾 | 高攻速阵容 |
| 虚空织者 | 周期性隐身+召唤分身 | AOE英雄 |
| 深渊暴君 | 狂暴后加速+禁锢塔防 | 控制流 |

## 五、洗练系统

### 5.1 基础洗练

| 属性 | 值 |
|------|---|
| 消耗 | 裂隙之尘 ×30 |
| 效果 | 重新随机**所有未锁定词条**的类型和数值 |
| 词条数量 | 不变 |
| 符文品质 | 不变 |

### 5.2 词条锁定

| 属性 | 值 |
|------|---|
| 消耗 | 符文封印 ×1（每锁定1条，每次洗练消耗） |
| 效果 | 洗练时保留该词条不变 |
| 上限 | 最多锁定 N-1 条（至少1条可洗） |

> **注意**：符文封印是**每次洗练时消耗**，不是一次性永久锁定。锁定2条词条洗练一次 = 消耗2枚符文封印 + 30裂隙之尘。这使得高锁定洗练成为高成本策略性选择。

### 5.3 进阶洗练：定向洗练

解锁条件：通关主线第 30 章（第600波）

| 属性 | 值 |
|------|---|
| 消耗 | 裂隙之尘 ×80 + 深渊结晶 ×1 |
| 效果 | 选择一个词条类别（基础/特殊），只从该类别中重随 |

### 5.4 词条升品

解锁条件：通关主线第 40 章（第800波）

| 属性 | 值 |
|------|---|
| 消耗 | 裂隙之尘 ×100 + 深渊结晶 ×3 |
| 效果 | 选择一条词条，按下表概率提升其品质（白→绿→蓝→紫→橙→红） |
| 失败 | **词条品质降低一级**（最低降至白色），材料消耗 |

**升品成功率（按当前词条品质）**：

| 当前品质 | 成功率 | 说明 |
|---------|--------|------|
| 白色 → 绿色 | 100% | 必定成功 |
| 绿色 → 蓝色 | 80% | |
| 蓝色 → 紫色 | 50% | |
| 紫色 → 橙色 | 25% | |
| 橙色 → 红色 | 10% | 风险极高，失败降回紫色 |

> **设计意图**：低品质词条容易升品，鼓励玩家尝试；高品质词条升品风险极大（失败还会降品），使"全红词条"成为真正的终极追求。

## 六、材料体系

### 6.1 新增材料

| 材料 | 图标色 | 获取方式 | 用途 |
|------|--------|---------|------|
| **裂隙之尘** | `{160,120,200}` | 深渊裂隙副本掉落 | 基础洗练 |
| **符文封印** | `{40,200,160}` | 深渊裂隙副本掉落（精英波/BOSS波）；试练塔（每5层奖励×2）；世界BOSS伤害阈值奖励 | 洗练时锁定词条 |
| **深渊结晶** | `{200,60,255}` | 分解橙色/红色品质符文；噩梦难度BOSS低概率掉落 | 定向洗练、词条升品 |

> **符文封印**：古老的封印符咒，刻在符文词条上可将其锁定，使其在洗练时不被改变。

**符文封印产出明细**：

| 来源 | 每次获得 | 日均估算 |
|------|---------|---------|
| 深渊裂隙副本：精英波/BOSS波概率掉落 | ×1~2 | 约1~4（3次副本） |
| 试练塔：每5层奖励 | ×2 | 取决于推塔进度（一次性） |
| 世界BOSS：500万/5000万/50亿伤害阈值 | ×1/×1/×2 | 最多4×4=16（理想情况） |
| 分解紫色以上符文 | ×1（紫）/×2（橙）/×3（红） | 约0~2 |

> **设计意图**：符文封印产出分散在多个渠道，不能单纯靠刷一个副本获得，促使玩家参与多种内容。日均获得约 3~6 枚，支撑每天 2~3 次锁定洗练。

### 6.2 符文分解

玩家可分解不需要的符文，获得材料：

| 符文品质 | 分解获得 |
|---------|---------|
| 白色 | 裂隙之尘 ×5 |
| 绿色 | 裂隙之尘 ×10 |
| 蓝色 | 裂隙之尘 ×20 |
| 紫色 | 裂隙之尘 ×40 + 符文封印 ×1 |
| 橙色 | 裂隙之尘 ×60 + 符文封印 ×2 + 深渊结晶 ×1 |
| 红色 | 裂隙之尘 ×100 + 符文封印 ×3 + 深渊结晶 ×3 |

### 6.3 符文背包

- 背包容量：**50 个**（可用裂隙之尘扩容，每次 +10，最大100）
- 满背包时新获得的符文自动进入"待处理"队列（最多5个）
- 超出后必须分解或装备才能继续获得

## 八、数值平衡

### 8.1 符文强度对标

以"全身紫色符文、中等词条"为基准（中期玩家）：

| 属性 | 3个符文合计约 | 相当于 |
|------|-------------|--------|
| 攻击力 | +15~25% | 约2000级装备的攻击加成 |
| 暴击率 | +8~15% | 淬炼3~4条中等暴击 |
| 暴击伤害 | +15~30% | 头盔500~800级 |

**设计原则**：符文是**锦上添花**，不是**核心战力来源**。全红完美词条大约提升 30~40% 总输出。

### 8.2 日均产出估算

| 活动 | 裂隙之尘 | 符文封印 | 其他 |
|------|---------|--------|------|
| 3次深渊裂隙（普通） | ~120 | 1~3 | 3个符文 |
| 3次深渊裂隙（噩梦） | ~200 | 2~4 | 3个符文 + 偶尔深渊结晶 |
| 世界BOSS（4次） | — | 4~16 | — |
| 分解多余符文（约2个蓝色） | ~40 | 0~1 | — |
| **日均合计** | **~160~240** | **~5~10** | — |
| **日均洗练** | 普通洗练 5~8次 | 锁定洗练 3~5次 | — |

**完美毕业周期**：一个英雄3个槽全部完美 ≈ 数周持续刷取（视运气）。

## 九、与现有系统的关系

### 9.1 不冲突

| 现有系统 | 符文系统 | 关系 |
|---------|---------|------|
| 4件装备（固定属性） | 3个符文（随机词条） | 互补，装备提供基础战力，符文提供个性化 |
| 淬炼系统 | 洗练系统 | 淬炼针对装备本体，洗练针对符文，资源独立 |
| 锻魂铁/暗影精粹 | 裂隙之尘/深渊结晶 | 完全不同的货币体系，不争夺资源 |

### 9.2 新增副本的定位

```
资源副本4种 ──────── 刷货币（冥晶/噬魂石/锻魂铁/宝箱）
试练塔 ─────────── 刷招募令
世界BOSS ────────── 刷限定招募
深渊裂隙（新增）──── 刷符文+洗练材料
```

|------|------|
| 符文升级 | 消耗同品质符文作为经验，提升符文等级（词条数值+5%/级） |
| 符文觉醒 | 红色符文满级后觉醒，解锁第5条专属词条 |
| 赛季符文 | 限时系列符文（如春节系列），赛季结束后不再掉落 |
| 符文交易 | 玩家之间交换符文（需要社交系统支持） |
| 深渊裂隙排行 | 按通关时间排名，奖励深渊结晶 |
