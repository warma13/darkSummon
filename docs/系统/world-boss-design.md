# 世界BOSS设计文档

## 一、概述

**世界BOSS** 是副本系统中的新玩法模式，玩家需要面对一个**常驻的超高血量BOSS**，同时应对不断涌来的小怪潮。BOSS拥有三种主动技能——**束缚、召唤精英、销毁英雄**——给玩家制造持续压力。局内低星（合成星级）的英雄更容易被BOSS针对，激励玩家在战斗中积极合成高星英雄。

**核心体验**：在BOSS技能的持续威胁下管理阵容，通过合成提高英雄星级来降低被针对的概率。

## 二、入口与解锁

| 项目 | 说明 |
|------|------|
| 入口位置 | 副本页面，新增第三张卡片"世界BOSS" |
| 解锁条件 | 通关试练塔第3塔（第30层） |
| 每日次数 | 免费1次 + 广告3次 = 每日最多4次 |
| 排行榜 | 按最高通关波次排名 |

## 三、战斗规则

### 3.1 基础参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 总波数 | 20波 | 与资源副本一致 |
| 每波小怪数 | 20只 | 固定数量 |
| BOSS出现时机 | 第1波开始即出现 | 常驻场上，不随波次刷新 |
| 超限触发 | 场上怪物 > 20只 | 10秒倒计时，超时失败 |
| 波次间隔 | 25秒 | 与试练塔一致 |
| 自动推波 | 是 | 击杀当波所有小怪后自动下一波 |
| 暗魂初始值 | 同主线 Config.INITIAL_DARK_SOUL | |

> **超限说明**：这里的"超限"是指场上怪物数量超过上限（20只），而非额外波次。超限后进入10秒倒计时，倒计时归零则判定失败。玩家需要在倒计时内将场上怪物清到20只以下。

### 3.2 BOSS 设定

| 属性 | 值 | 说明 |
|------|-----|------|
| 名称 | 深渊主宰 | 世界BOSS专属 |
| HP | math.maxinteger（不可击杀） | 全服统一，无限血量 |
| DEF | 50,000 | 全服统一，固定值 |
| 移动速度 | 12（极慢） | BOSS不以到达终点为威胁 |
| 体型 | size = 28 | 超大体型 |
| 形状 | diamond | 菱形 |
| 免疫 | 冰冻、处决、减速效率50%、眩晕时间50% | 对齐 Config.BOSS_BALANCE |

**全服统一难度**：BOSS的DEF对所有玩家相同，不随主线进度缩放。

**BOSS不可击杀**——血量为 `math.maxinteger`，从第1波常驻到战斗结束。玩法核心是在20波内尽可能多地对BOSS输出伤害，奖励按累计伤害量结算。

## 四、BOSS技能系统

BOSS拥有三种主动技能，按独立冷却轮流释放。技能目标选择基于**局内合成星级**（★1~★5）。

> **局内合成星级说明**：战斗中英雄的星级（★1~★5），通过拖拽2个同星同类英雄合成升星获得。与局外养成星级（1~30星）无关。★1最弱最容易被针对，★5最强最不容易被针对。

### 4.1 束缚（Shackle）

| 参数 | 值 |
|------|-----|
| 冷却时间 | 15秒 |
| 首次释放 | 第20秒 |
| 效果 | 选中1名英雄，束缚5秒 |
| 被束缚状态 | 无法移动、无法攻击、无法释放技能 |
| 视觉效果 | 锁链缠绕特效 + 英雄变灰 |

**目标选择权重（按局内合成星级）**：

| 局内星级 | 被束缚权重 | 说明 |
|----------|-----------|------|
| ★1 | 5（最高） | 未合成，最容易被束缚 |
| ★2 | 4 | |
| ★3 | 3 | |
| ★4 | 2 | |
| ★5 | 1（最低） | 满星，最不容易被束缚 |
| 主角（暗影君主） | 1（固定最低） | 不参与合成，固定最低权重 |

**计算公式**：
```lua
被束缚概率 = 该英雄权重 / 所有存活且未被束缚的英雄权重之和
```

### 4.2 召唤精英（Summon Elite）

| 参数 | 值 |
|------|-----|
| 冷却时间 | 30秒 |
| 首次释放 | 第30秒 |
| 效果 | 召唤精英怪 |

**召唤数量递增**：

| 第N次召唤 | 精英数量 |
|-----------|---------|
| 第1次 | 1只 |
| 第2次 | 2只 |
| 第3次 | 3只 |
| 第4次+ | min(N, 5)只 |

**精英怪规格**：
- 从当前主题的可用角色池中随机选择（不含 minion）
- HP = 该波普通怪HP × 3.0
- 附带T1词缀（tough/swift/regen/berserk随机一个）
- 后期波次（16+）附带T2词缀
- 击杀精英掉落额外货币奖励

### 4.3 销毁英雄（Annihilate）

| 参数 | 值 |
|------|-----|
| 冷却时间 | 45秒 |
| 首次释放 | 第60秒 |
| 效果 | 直接移除1名英雄（本场战斗内永久退场） |
| 触发提示 | 释放前3秒全屏警告："深渊主宰正在凝聚毁灭之力！" |

**目标选择权重（按局内合成星级）**：

| 局内星级 | 被销毁权重 | 说明 |
|----------|-----------|------|
| ★1 | 6（最高） | 未合成，最容易被销毁 |
| ★2 | 4 | |
| ★3 | 2 | |
| ★4 | 1 | |
| ★5 | 0.5（极低） | 满星，几乎不会被销毁 |
| 主角（暗影君主） | 0（免疫） | 永远不会被销毁 |

**重要规则**：
- 主角"暗影君主"**免疫销毁**（权重为0，永远不会被选中）
- 被销毁的英雄仅在**本场战斗内**退场，不影响永久数据
- 如果场上只剩主角1人，销毁技能改为对主角施加5秒束缚
- ★5英雄虽然权重极低（0.5），但不免疫，仍有小概率被销毁

**视觉效果**：
1. BOSS蓄力3秒（全屏红色警告文字）
2. 目标英雄头顶出现骷髅标记
3. 暗影射线击中目标
4. 目标英雄碎裂消散动画
5. 底部提示："[英雄名] 被深渊主宰销毁！"

### 4.4 技能释放时间轴（示例）

```
0s ─── 战斗开始，BOSS出现
20s ── 束缚 #1（1名英雄被束缚5秒）
30s ── 召唤精英 #1（1只精英怪）
35s ── 束缚 #2
50s ── 束缚 #3
60s ── 召唤精英 #2（2只精英怪）
60s ── ⚠ 销毁 #1（1名英雄退场）
65s ── 束缚 #4
80s ── 束缚 #5
90s ── 召唤精英 #3（3只精英怪）
95s ── 束缚 #6
105s ─ ⚠ 销毁 #2（又1名英雄退场）
...
```

### 4.5 战术意义

局内合成星级作为目标选择依据，形成以下策略：
- **积极合成**：尽快将★1英雄合成为★3+，降低被束缚/销毁的概率
- **★1英雄是消耗品**：刚召唤的★1英雄极易被BOSS针对，要快速合成
- **★5英雄是核心输出**：几乎不会被束缚/销毁，是持久战的关键
- **合成节奏**：不能只攒暗魂不召唤，需要维持一定数量的英雄在场

## 五、小怪系统

### 5.1 波次结构

每波刷新 **20只小怪**，不含BOSS（BOSS从第1波开始常驻）。

### 5.2 角色池

使用与主线相同的 10种角色 × 5种主题 系统，主题随波次轮换：

| 波次范围 | 主题 |
|----------|------|
| 1~4波 | 亡灵墓地 |
| 5~8波 | 熔岩地狱 |
| 9~12波 | 幽暗森林 |
| 13~16波 | 冰霜冻土 |
| 17~20波 | 虚空深渊 |

### 5.3 角色解锁节点

复用主线的 `ROLE_UNLOCK_WAVE` 逻辑，按全局波次映射：

| 角色 | 解锁波次 |
|------|---------|
| minion, infantry | 第1波 |
| tank, assassin | 第3波 |
| dodger | 第5波 |
| support | 第7波 |
| splitter | 第9波 |
| blinker | 第11波 |
| special | 第13波 |

### 5.4 小怪难度曲线

**全服统一难度**：BOSS和小怪的数值对所有玩家相同。

**HP缩放**（以固定等效关卡为基准）：

```lua
-- 固定基准等效关卡（全服统一）
local BASE_STAGE_EQUIV = 500  -- 等效主线第500关

-- 每波递增
local waveStage = BASE_STAGE_EQUIV * (1 + (wave - 1) * 0.12)  -- 每波+12%

local hpScale = CalcHPScale(waveStage)
```

**速度缩放**：
```lua
local speedScale = math.min(1.0 + (wave - 1) * 0.02, 1.8)  -- 上限1.8倍
```

### 5.5 词缀系统

| 波段 | 词缀规则 |
|------|---------|
| 1~5波 | 无词缀 |
| 6~10波 | 每波1只T1词缀精英 |
| 11~15波 | 每波2只T1精英 + 1只T2精英 |
| 16~20波 | 每波3只T2精英 + 1只T3精英 |

## 六、奖励系统

### 6.1 BOSS伤害奖励

奖励按**对BOSS造成的累计伤害量**发放。BOSS血量为最大值（不可击杀），玩家在20波内尽可能输出。

| 累计伤害 | 奖励 |
|---------|------|
| 10万 | 霜誓契约×1 |
| 20万 | 霜誓契约×1 |
| 100万 | 霜誓契约×1 |
| 500万 | 霜誓契约×1 |
| 1,000万 | 霜誓契约×2 |
| 2,000万 | 霜誓契约×2 |
| 5,000万 | 霜誓契约×2 |
| 1亿 | 霜誓契约×4 |
| 2亿 | 霜誓契约×4 |
| 5亿 | 霜誓契约×4 |

> **说明**：奖励为累计发放，达到更高档位时自动包含所有低档位奖励。战斗结束时按最终累计伤害一次性结算。全部达成共计 **霜誓契约×24**。

### 6.2 击杀掉落（实时）

小怪和精英的实时击杀掉落复用主线 `Config.KILL_DROP` 公式。

### 6.3 排行榜

|--------|---------|---------|
| 世界BOSS | 对BOSS造成的最高伤害量 | 复用 SCORE_OFFSET 编码 |

## 七、与现有系统的交互

### 7.1 局内合成策略

世界BOSS的束缚和销毁机制直接与局内合成星级挂钩：

| 局内星级 | 束缚权重 | 销毁权重 | 策略建议 |
|----------|---------|---------|---------|
| ★1 | 5 | 6 | 刚召唤，极易被针对，尽快合成 |
| ★2 | 4 | 4 | 仍然危险，继续合成 |
| ★3 | 3 | 2 | 中等风险，可作为过渡 |
| ★4 | 2 | 1 | 较安全 |
| ★5 | 1 | 0.5 | 核心战力，几乎免疫 |
| 主角 | 1 | 0（免疫） | 永远不会被销毁 |

### 7.2 上阵与合成

- 战斗中通过召唤+合成不断生产高星英雄
- BOSS的销毁技能会逐步减少阵容，低星英雄首当其冲
- 需要在"攒暗魂等合成"和"及时召唤补充火力"之间做取舍
- ★5英雄是持久战核心，优先保护

### 7.3 装备系统

- 世界BOSS战斗中，装备效果正常生效
- 高等级装备的属性加成是抵抗BOSS高DEF的关键

### 7.4 元素相性

- BOSS使用"虚空"主题，暗属性
- 雷/毒元素英雄对BOSS有额外增伤（对齐 THEME_ELEMENT_RESIST.void）
- 暗属性英雄对BOSS伤害降低

## 十二、开发优先级

1. **P0 - 核心战斗**：BOSS常驻 + 20波小怪 + 超限判定（20只上限）
2. **P0 - 束缚技能**：按局内合成星级权重选目标 + 束缚状态
3. **P0 - 销毁技能**：按局内合成星级权重选目标 + 英雄退场
4. **P1 - 召唤精英**：递增数量 + 词缀
5. **P1 - 奖励系统**：波次奖励 + 排行榜
6. **P2 - UI 完善**：BOSS状态栏、技能预警、销毁动画
