# 试练塔系统设计

> 试练塔是独立于主线关卡的挑战副本，以"塔"为单位逐层攻克。
> 每座塔包含 10 层，通关一座塔直发 10 招募令（= 一次十连），是推进进度的核心驱动力之一。
> 试练塔**不产出魂石**，只给进阶石 + 金币（每层）和招募令（每塔）。

---

## 1. 核心结构

| 概念 | 说明 |
|------|------|
| **塔** | 试练塔的基本挑战单元，每座塔包含 10 层 |
| **层** | 塔内的单次战斗关卡，每层有独立的敌人配置和首通奖励 |
| **通塔** | 通过一座塔的全部 10 层，触发通塔奖励 |
| **当前层** | 玩家可挑战的最新层（= 历史最高通关层 + 1） |

### 数据结构

```lua
-- HeroData 中新增字段
HeroData.towerData = {
    currentFloor = 1,      -- 当前可挑战层（1-based，全局累计）
    clearedFloors = 0,     -- 已通关总层数
    claimedTowers = 0,     -- 已领取通塔奖励的塔数
}
```

### 层号与塔号的关系

```
塔号 = math.ceil(层号 / 10)
塔内层号 = ((层号 - 1) % 10) + 1

示例:
  第 1 层 → 第 1 塔 第 1 层
  第 10 层 → 第 1 塔 第 10 层
  第 11 层 → 第 2 塔 第 1 层
  第 100 层 → 第 10 塔 第 10 层
```

## 2. 奖励设计

### 2.1 每层首通奖励（一次性）

| 塔号 | 每层进阶石(噬魂石) | 每层金币(冥晶) | 难度 |
|------|-------------------|---------------|------|
| 第 1 塔 | 5 | 200 | 入门 |
| 第 2 塔 | 8 | 350 | 入门 |
| 第 3 塔 | 10 | 450 | 简单 |
| 第 4 塔 | 12 | 550 | 简单 |
| 第 5 塔 | 14 | 650 | 简单 |
| 第 6 塔 | 16 | 750 | 普通 |
| 第 7 塔 | 18 | 850 | 普通 |
| 第 8 塔 | 20 | 1,000 | 普通 |
| 第 9 塔 | 22 | 1,100 | 普通 |
| 第 10 塔 | 25 | 1,200 | 中等 |
| 第 11~19 塔 | 28~45 (线性增长) | 1,400~2,200 (线性增长) | 中等~困难 |
| 第 20 塔起 | 50+ (递增) | 2,500+ (递增) | 极难 |

**计算公式**（第 n 塔）：
```lua
if n <= 10 then
    -- 从配置表查询
elseif n <= 19 then
    -- 线性插值: 28 + (n - 11) * 2 进阶石
    -- 线性插值: 1400 + (n - 11) * 100 金币
else
    -- 50 + (n - 20) * 5 进阶石
    -- 2500 + (n - 20) * 200 金币
end
```

### 2.2 通塔奖励（每座塔 10 层全通后，一次性）

- 每座塔通关后直发 **10 招募令**
- 无需手动领取，通关第 10 层时自动发放
- 弹窗展示通塔成就 + 招募令获得

### 2.3 货币映射

| 设计文档名称 | 游戏内货币 | Config 字段 |
|-------------|-----------|------------|
| 进阶石 | 噬魂石 | `devour_stone` |
| 金币 | 冥晶 | `nether_crystal` |
| 招募令 | 虚空契约 | `void_pact` |

## 3. 难度曲线

### 3.1 阶段划分

| 阶段 | 塔号 | 对应层数 | 难度描述 | 预期推进速度 |
|------|------|---------|---------|-------------|
| 新手期 | 第 1~10 塔 | 1~100 层 | 低难度，引导新玩家 | 首周可全部通关 |
| 成长期 | 第 11~19 塔 | 101~190 层 | 中等，需要基础养成 | 每座塔 1~2 周 |
| 挑战期 | 第 20 塔起 | 191+ 层 | 每座塔 ≈ 100 关主线难度 | 数周甚至更久/塔 |

### 3.2 敌人配置

试练塔的敌人基于主线怪物系统（Config.THEMES），但有独立的数值缩放：

```lua
-- 试练塔 HP 缩放公式
local baseHP = role.baseHP
local floorMult = 1.0 + (globalFloor - 1) * 0.15     -- 每层 +15% HP
local towerMult = 1.0 + (towerNum - 1) * 0.50         -- 每塔额外 +50%
local finalHP = baseHP * floorMult * towerMult

-- 试练塔 DEF 缩放
local finalDEF = baseDEF * (1.0 + (globalFloor - 1) * 0.08)
```

- 每层出 1 波怪（简化，区别于主线的 20 波制）
- 第 10 层（塔顶）为 BOSS 层，出对应主题 BOSS
- 主题按塔号循环：`themeIndex = ((towerNum - 1) % 5) + 1`

### 3.3 战斗规则

| 规则 | 说明 |
|------|------|
| 阵容 | 使用玩家当前上阵英雄（与主线共享） |
| 英雄等级/星级 | 使用当前养成状态 |
| 战斗时限 | 每层 60 秒，超时视为失败 |
| 失败惩罚 | 无惩罚，可无限重试 |
| 通关条件 | 在时限内击杀所有敌人 |
| 进度保存 | 通关后自动保存，下次挑战从下一层开始 |

- 未解锁：显示奖励内容 + "需通过前一层"

## 5. 战斗流程

```
玩家点击"挑战"
  ↓
进入试练塔战斗（使用当前阵容 + 养成状态）
  ↓
自动生成本层敌人（基于层号和主题）
  ↓
战斗开始（60秒倒计时）
  ├─ 胜利 → 发放首通奖励 → 更新进度
  │         ├─ 如果是第 10 层 → 通塔弹窗 + 发 10 招募令
  │         └─ 自动进入下一层（或返回试练塔界面）
  └─ 失败 → 显示失败面板 → 可重试或返回
```

## 7. 与现有系统的关联

| 系统 | 关联方式 |
|------|---------|
| 英雄养成 | 试练塔使用当前英雄等级/星级/进阶/觉醒状态 |
| 招募系统 | 通塔奖励直发虚空契约(招募令)，驱动招募循环 |
| 货币系统 | 每层产出噬魂石(进阶石) + 冥晶(金币) |
| 成就系统 | 通关 30/50/80/100 层触发成就魂石奖励 |
| 主线关卡 | 独立于主线，互不影响，但共享英雄阵容 |
