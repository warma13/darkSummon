## 1. 系统概览

宝箱系统是局外养成的资源产出渠道之一，通过通关关卡获取宝箱，开箱获得货币和英雄碎片。

```
通关关卡 → 获得宝箱 → 开箱 → 货币 / 碎片
                         ↓
                     累积积分 → 里程碑奖励（更多宝箱/货币）
```

### 核心循环

- **产出端**: 通关掉落宝箱（按关卡阶段递增品质）
- **消耗端**: 开箱消耗宝箱库存
- **奖励**: 冥晶、噬魂石、暗影精华 + 各稀有度英雄碎片
- **积分系统**: 开箱累积积分，达到里程碑领取额外奖励

## 2. 宝箱品质

共 5 种品质宝箱，从低到高：

| 品质 | 名称 | Emoji | 积分/个 | 主要产出 |
|------|------|-------|---------|----------|
| 木 | 朽木宝箱 | 🪵 | 1 | 冥晶 20~50 |
| 铜 | 青铜宝箱 | 🥉 | 10 | 噬魂石 5~15 |
| 金 | 黄金宝箱 | 🥇 | 20 | 噬魂石 15~40 |
| 铂 | 铂金宝箱 | 💎 | 50 | 噬魂石 30~80 + SR碎片 |
| 钻 | 钻石宝箱 | 👑 | — | 暗影精华 10~30 + SSR碎片 |

### 初始赠送

新玩家默认获得：
- 朽木宝箱 ×10
- 青铜宝箱 ×3

## 3. 获取途径

### 通关掉落

| 条件 | 掉落 |
|------|------|
| 每关通关 | 朽木宝箱 ×3 |
| 每 5 关通关 | 青铜宝箱 ×1 |
| 每 10 关通关 | 黄金宝箱 ×1 |

### 积分里程碑奖励

达到积分阈值自动领取（详见下方积分系统）。

## 4. 开箱掉落

每次开箱可批量开启（最多 10 个同类宝箱），每个宝箱独立判定掉落。

### 详细掉落表

**朽木宝箱**
| 掉落物 | 数量 | 概率 |
|--------|------|------|
| 冥晶 | 20~50 | 100% |
| N 碎片 | 1~2 | 5% |

**青铜宝箱**
| 掉落物 | 数量 | 概率 |
|--------|------|------|
| 噬魂石 | 5~15 | 100% |
| R 碎片 | 1~3 | 7% |
| SR 碎片 | 1~2 | 3% |

**黄金宝箱**
| 掉落物 | 数量 | 概率 |
|--------|------|------|
| 噬魂石 | 15~40 | 100% |
| SR 碎片 | 2~4 | 20% |
| SSR 碎片 | 1~2 | 3% |

**铂金宝箱**
| 掉落物 | 数量 | 概率 |
|--------|------|------|
| 噬魂石 | 30~80 | 100% |
| SR 碎片 | 3~5 | 100% |
| SSR 碎片 | 2~4 | 10% |

**钻石宝箱**
| 掉落物 | 数量 | 概率 |
|--------|------|------|
| 暗影精华 | 10~30 | 100% |
| SSR 碎片 | 3~6 | 100% |
| UR 碎片 | 1~3 | 33% |

### 碎片掉落规则

- 碎片从对应稀有度的招募池中随机选择一个英雄
- 碎片自动累积，达到解锁阈值自动解锁英雄

## 5. 积分进度

开箱累积积分，每轮总目标 500 分。

### 里程碑奖励

| 积分 | 奖励 |
|------|------|
| 50 | 青铜宝箱 ×2 |
| 150 | 黄金宝箱 ×2 |
| 300 | 铂金宝箱 ×1 + 虚空契约 ×3 |
| 500 | 钻石宝箱 ×1 + 暗影精华 ×20 |

### 积分规则

- 每个宝箱开启贡献固定积分（朽木1分、青铜10分、黄金20分、铂金50分）
- 钻石宝箱不贡献积分
- 达到里程碑自动领取，已领取的里程碑标记为 ✓
- 支持积分重置（新一轮），重置后里程碑可重新领取

## 6. 数据存储

宝箱数据挂载在 `HeroData.chestData` 上，随存档保存/加载。

```lua
HeroData.chestData = {
    inventory = {           -- 各品质宝箱库存
        wood = 10,
        bronze = 3,
        gold = 0,
        platinum = 0,
        diamond = 0,
    },
    score = 0,              -- 当前轮积分
    claimedMilestones = {}, -- 已领取的里程碑索引 { [1]=true, [2]=true, ... }
}
```

## 7. UI 界面

### 页面结构

```
┌──────────────────────────────┐
│          宝箱（标题）          │
├──────────────────────────────┤
│  积分进度条  50/500  ●─●─○─○  │
├──────────────────────────────┤
│                              │
│      [宝箱名称 + 品质色]      │
│      [SSR概率: X%]           │
│                              │
│         ┌────────┐           │
│         │  大图标  │           │
│         └────────┘           │
│          X 数量              │
│                              │
│      掉落说明（逐行）         │
│                              │
├──────────────────────────────┤
│      [ 打开 N 个宝箱 ]       │
├──────────────────────────────┤
│  🪵  🥉  🥇  💎  👑          │
│  X10 X3  X0  X0  X0         │
└──────────────────────────────┘
```

### 交互说明

- 底部选择栏切换宝箱品质，有库存的显示红点
- 开箱按钮一次最多开 10 个，库存不足时置灰
- 开箱后弹出结果弹窗，展示获得的货币和碎片
