# 咸鱼之王 — 数值与经济系统参考

> 本文档梳理咸鱼之王的核心数值体系和经济循环，作为 Dark Merge TD 数值设计的对标参考。
> 数据来源：公开攻略、社区整理，版本截至 2025 年初，仅供设计参考。

---

## 1. 英雄成长体系概览

咸鱼之王的英雄成长由 **等级 × 升星 × 觉醒** 三个独立维度组成，互相乘算：

```
最终战力 = 基础属性 × 等级加成 × 升星加成 × 觉醒加成
```

| 维度 | 上限 | 主要消耗 | 提升幅度 |
|------|------|---------|---------|
| 等级 | Lv6000 | 金币 + 进阶石 | 巨大（数百倍） |
| 升星 | 30 星（6 色阶） | 英雄碎片 | 每星 +10%，进阶 +40% |
| 觉醒 | 4 阶段 | 自动解锁（星级触发） | 技能强化 |

## 2. 等级系统

### 2.1 基本规则

- **等级上限**: Lv6000
- **升级消耗**: 金币（随等级递增）
- **进阶门槛**: 到达特定等级后必须消耗 **进阶石** 才能继续升级

### 2.2 进阶石门槛

等级被划分为多个区间，每个区间需要消耗进阶石突破：

| 等级区间 | 进阶石消耗（约） | 说明 |
|---------|-----------------|------|
| Lv1 → 100 | 0 | 免费升级 |
| Lv100 突破 | 20 | 第一个门槛 |
| Lv100 → 300 | 金币 | 正常升级 |
| Lv300 突破 | ~50 | |
| Lv300 → 500 | 金币 | |
| Lv500 突破 | ~100 | |
| Lv500 → 700 | 金币 | |
| Lv700 突破 | ~200 | |
| Lv700 → 900 | 金币 | |
| Lv900 → 1100 | ~3,000 | 消耗开始陡增 |
| Lv1100 → 1300 | ~5,000 | |
| Lv1300 → 1500 | ~8,000 | |
| Lv1500 → 2000 | ~2 万 | |
| Lv2000 → 2500 | ~5 万 | |
| Lv2500 → 3000 | ~10 万 | |
| Lv3000 → 3500 | ~20 万 | |
| Lv3500 → 4000 | ~35 万 | |
| Lv4000 → 5000 | ~50 万 | |
| Lv5000 → 6000 | ~84 万 | 后期天文数字 |

**设计意图**：
- 前期（Lv1-300）金币充足即可升级，让玩家快速感受到成长
- 中期（Lv300-1500）进阶石成为瓶颈，引导玩家参与更多玩法获取
- 后期（Lv1500+）进阶石消耗急剧膨胀，形成长线追求

### 2.3 升级金币消耗

升级金币消耗随等级线性/二次增长，前期非常便宜，后期金币同样大量消耗但**进阶石才是真正瓶颈**。

## 3. 升星系统

### 3.1 星级结构

30 星分为 **6 个色阶**，每个色阶 5 颗星：

| 色阶 | 星级范围 | 颜色 | 每星属性加成 | 色阶进阶加成 |
|------|---------|------|------------|------------|
| 第 1 阶 | 1 ~ 5 星 | 黄星 ⭐ | +10% | +40%（5→6 星进阶） |
| 第 2 阶 | 6 ~ 10 星 | 紫星 💜 | +10% | +40%（10→11 星进阶） |
| 第 3 阶 | 11 ~ 15 星 | 橙星 🟠 | +10% | +40%（15→16 星进阶） |
| 第 4 阶 | 16 ~ 20 星 | 红星 🔴 | +10% | +40%（20→21 星进阶） |
| 第 5 阶 | 21 ~ 25 星 | 皇冠 👑 | +10% | +40%（25→26 星进阶） |
| 第 6 阶 | 26 ~ 30 星 | 紫晶 💎 | +10% | — |

**满星总加成**: 30 × 10% + 5 × 40% = **500%**（即属性变为基础的 6 倍）

### 3.2 碎片消耗

每颗星消耗的碎片数量按色阶递增：

| 色阶 | 每星碎片 | 该阶总碎片（5 星） |
|------|---------|------------------|
| 黄星 | 8 | 40 |
| 紫星 | 40 | 200 |
| 橙星 | 80 | 400 |
| 红星 | 200 | 1,000 |
| 皇冠 | 400 | 2,000 |
| 紫晶 | 400 | 2,000 |

**单英雄满星总碎片**: 40 + 200 + 400 + 1,000 + 2,000 + 2,000 = **5,640 碎片**

### 3.3 碎片获取途径

| 途径 | 产出量 | 说明 |
|------|-------|------|
| 招募（抽卡） | 主要来源 | 重复英雄自动转为碎片 |
| 宝箱系统 | 中等来源 | 5 种宝箱，高级宝箱概率出碎片 |
| 商店购买 | 少量 | 金砖/钻石兑换 |
| 活动奖励 | 波动性 | 特定活动期间大量产出 |
| 咸将令（赛季通行证） | 稳定来源 | 每赛季固定产出 |

## 4. 觉醒系统

### 4.1 觉醒条件

觉醒不消耗额外资源，达到指定星级后自动解锁：

| 觉醒阶段 | 触发星级 | 效果 |
|---------|---------|------|
| 觉醒 1 | 22 星（皇冠 2 星） | 解锁/强化被动技能 |
| 觉醒 2 | 25 星（皇冠 5 星） | 进一步强化 |
| 觉醒 3 | 27 星（紫晶 2 星） | 技能质变 |
| 觉醒 4 | 30 星（紫晶 5 星，满星） | 最终形态 |

### 4.2 设计意图

- 觉醒作为升星的 **里程碑奖励**，不增加额外资源压力
- 每次觉醒给予显著的质变感（新技能效果、视觉变化）
- 22 星（皇冠 2 星）门槛较高，形成中后期目标

## 5. 经济系统

### 5.1 货币体系

| 货币 | 获取方式 | 消耗用途 | 定位 |
|------|---------|---------|------|
| **金币** | 挂机、关卡、宝箱 | 英雄升级 | 基础货币，前期充裕后期紧张 |
| **金砖** | 活动、商城、成就 | 黑市购物、洗练 | 中级货币 |
| **钻石** | 充值、活动 | 抽卡、加速 | 付费货币 |
| **英雄碎片** | 抽卡、宝箱、活动 | 英雄升星 | 角色专属资源 |
| **进阶石** | 关卡奖励、活动 | 等级进阶突破 | 成长瓶颈资源 |
| **觉醒材料** | 自动获得（升星触发） | 无额外消耗 | 里程碑奖励 |

### 5.2 核心经济循环

```
┌────────────────────────────────────────────┐
│              玩家行为循环                     │
│                                            │
│  挂机/打关 → 金币 → 升级英雄 → 战力提升       │
│      ↓                        ↓            │
│  推更高关卡 ← ← ← ← ← ← ← ←┘            │
│      ↓                                     │
│  获取进阶石 → 突破等级上限 → 更高战力          │
│      ↓                                     │
│  获取碎片/钻石 → 抽卡/升星 → 阵容强化         │
│      ↓                                     │
│  推更高关卡 (循环)                            │
└────────────────────────────────────────────┘
```

### 5.3 资源产出节奏

**每日稳定产出**：
- 金币：挂机收益 + 日常任务 + 竞技场
- 碎片：每日免费抽卡（少量）
- 进阶石：日常任务（少量）

**每周活动轮换**：
咸鱼之王采用 **周活动轮换** 机制，引导玩家周期性消耗不同资源：

| 活动周 | 核心内容 | 资源方向 |
|-------|---------|---------|
| 招募周 | 抽卡加倍/保底 | 碎片大量产出 |
| 宝箱周 | 宝箱奖励加倍 | 金币/碎片/道具 |
| 黑市周 | 黑市折扣 | 金砖消耗 |
| 消耗活动 | 消耗累计奖励 | 引导囤积资源集中消耗 |

### 5.4 宝箱系统

5 种宝箱，按稀有度递增：

| 宝箱 | 内容 | 获取方式 |
|------|------|---------|
| 木质宝箱 | 少量金币 | 挂机、日常 |
| 铁质宝箱 | 金币 + 小概率碎片 | 关卡掉落 |
| 银质宝箱 | 金币 + 碎片 | 活动、成就 |
| 金质宝箱 | 大量碎片 + 道具 | 活动、付费 |
| 钻石宝箱 | 英雄 + 大量资源 | 特殊活动 |

## 6. 养成周期估算（F2P 视角）

| 里程碑 | 预估时间 | 说明 |
|-------|---------|------|
| 首个英雄 Lv100 | 1-2 天 | 新手引导快速达成 |
| 首个英雄 10 星（紫星满） | 2-3 周 | 碎片逐步积累 |
| 首个英雄 Lv1000 | 1-2 月 | 进阶石成为瓶颈 |
| 首个英雄 20 星（红星满） | 2-3 月 | 碎片需求大幅增长 |
| 首个英雄觉醒 1（22 星） | 3-4 月 | 皇冠阶碎片昂贵 |
| 首个英雄 Lv3000 | 4-5 月 | 进阶石消耗急剧增长 |
| 首个英雄满星（30 星） | ~5 月+ | 紫晶阶碎片总需 2000 |
| 首个英雄 Lv6000 | 6 月+ | 长线目标 |

> **关键结论**: F2P 玩家培养一个英雄到满配（Lv6000 + 30 星 + 满觉醒）大约需要 **4-6 个月**。
> 这保证了游戏的长线留存。
