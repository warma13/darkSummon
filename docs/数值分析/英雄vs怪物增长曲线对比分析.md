# 英雄 vs 怪物增长曲线对比分析

> 分析日期: 2026-04-22
> 基准英雄: UR 品质 (baseATK=3800, growthPct=15%)

---

## 一、英雄侧：六大乘区总倍率

英雄最终 ATK = baseATK × levelMult × advanceMult × starMult，技能/觉醒额外乘算到 DPS。

### 1. 等级倍率

公式: `levelMult = 1 + (level-1) × growthPct`

| 品质 | growthPct | Lv100 | Lv1000 | Lv3000 | Lv6000 |
|------|-----------|-------|--------|--------|--------|
| N    | 6%        | 6.94  | 60.94  | 180.94 | 360.94 |
| R    | 8%        | 8.92  | 80.92  | 240.92 | 480.92 |
| SR   | 10%       | 10.90 | 100.90 | 300.90 | 600.90 |
| SSR  | 12%       | 12.88 | 120.88 | 360.88 | 720.88 |
| UR   | 15%       | 15.85 | 150.85 | 450.85 | 900.85 |
| LR   | 18%       | 18.82 | 180.82 | 540.82 | 1080.82 |

**特点**: 纯线性增长，无加速，高品质优势固定为 growthPct 比值。

### 2. 进阶倍率

20 阶，每阶 +10%（乘算），即 `advanceMult = 1.10^N`。

| 进阶 | 解锁等级 | advanceMult |
|------|---------|-------------|
| 0    | -       | 1.00        |
| 5    | Lv700   | 1.61        |
| 10   | Lv2100  | 2.59        |
| 15   | Lv4000  | 4.18        |
| 20   | Lv5500  | 6.73        |

**特点**: 指数增长但有硬上限（20阶），Lv5500 后此乘区冻结。

### 3. 升星倍率

30 星，分 6 段（每段 5 星），段位切换 ×1.40:

| 段位     | 星级范围 | 每星倍率 | 段位进阶 |
|----------|---------|---------|---------|
| 黄星     | 1~5     | ×1.10   | ×1.40   |
| 紫星     | 6~10    | ×1.10   | ×1.40   |
| 橙星     | 11~15   | ×1.10   | ×1.40   |
| 红星     | 16~20   | ×1.10   | ×1.40   |
| 皇冠     | 21~25   | ×1.15   | ×1.40   |
| 紫晶     | 26~30   | ×1.15   | ×1.40   |

满星(30) 总倍率:
- 普通星(20星): 1.10^20 × 1.40^4 ≈ 6.73 × 3.84 ≈ 25.83
- 皇冠+紫晶(10星): 1.15^10 × 1.40^2 ≈ 4.05 × 1.96 ≈ 7.94
- **满星总计: 25.83 × 7.94 ≈ 104.5×**

**特点**: 乘区最大（×104.5），但 30 星满后冻结。

### 英雄 vs 怪物增长率对比

| 关卡段 | 英雄每 100 关增长 | 怪物每 100 关增长 | 差异 |
|--------|------------------|------------------|------|
| 0~500  | ×1.5~2.0/100关   | ×1.5~3.0/100关   | 基本匹配 |

**根因**: 英雄等级是线性增长（每级 +15% 基础），而怪物 HP 是分段加速增长。
英雄的主要增长来源在后期逐一触顶（升星30★→停、进阶20阶→停），增长率持续下降；
而怪物 HP 持续保持每 500 关 ×3 的增速。

## 四、问题诊断

### 问题 1: 3000 关后英雄增长远远追不上怪物 HP 🔴

- 英雄 3000→6000: 72,112 → 633,615 = **×8.8**
- 怪物 3000→6000: 180,000 → 35,000,000 = **×194**
- 差距从 ×2.5 扩大到 ×55

### 问题 2: 前 500 关英雄过度碾压 ⚠️

比值在前 300 关达到 2.0+，玩家缺乏挑战感，可能导致前期流失。

### 问题 3: 平衡窗口太窄 ⚠️

只有约 500~2500 关（依靠装备补差）的体验是"有挑战但能通过"的。

### 问题 4: 防御属性缩放 (ENEMY_SCALING) 起效太晚 ⚠️

critDmgReduce 在 300 关才开始，dmgBonus/elem/armorPen 在 500 关才开始。
前期英雄没有这些属性积累，减免形同虚设；后期英雄已经追不上怪物 HP，再加减免是雪上加霜。

### 问题 5: SPD 攻速加成上限过低（30%） ⚠️

`SPD_BONUS_MAX = 0.30`，倍率到 200 后攻速就封死。后期唯一能持续提升 DPS 的维度被硬卡。

### 问题 6: 装备/符文是隐性追赶机制，但被 ENEMY_SCALING 同步削减

装备后期提供的 dmgBonus/critDmg/elemDmg 经过软上限后约:
- Lv4000 装备: dmgBonus≈56%, critDmg≈84%, elemDmg≈56%
- 但同时被怪物的 ENEMY_SCALING 削减 30%~40%

## 五、优化建议

### 建议 1: 压平后半段怪物 HP 曲线 (优先级: 🔴高)

```lua
-- 当前后半段 (每段 ×3.0~3.3)
{ 3000, 3500,  180000,  550000 },
{ 3500, 4000,  550000, 1800000 },
{ 4000, 4500, 1800000, 6000000 },
{ 4500, 5000, 6000000, 15000000 },
{ 5000, 6000, 15000000, 35000000 },

-- 建议调整 (每段 ×1.8~2.0, 总增长 ×25 而非 ×194)
{ 3000, 3500,  180000,   350000 },
{ 3500, 4000,  350000,   700000 },
{ 4000, 4500,  700000,  1400000 },
{ 4500, 5000, 1400000,  2800000 },
{ 5000, 6000, 2800000,  5000000 },
```

调整后 6000 关 HP 缩放从 3500 万降到 500 万，与满配英雄 63 万倍率更匹配。

### 建议 2: 增加后期英雄成长维度 (优先级: 🔴高)

- **超越系统**: 满星后继续投入碎片，每次 ×1.05，无上限
- **转生系统**: 6000 级后重置等级，保留一定比例倍率加成
- **进阶扩展**: 提升至 30 阶，后 10 阶在 5000~6000 级解锁

### 建议 3: 缓和前期碾压体验 (优先级: 🟡中)

方案 A — 提高前期怪物 HP 基础值:
```lua
{   2,  100,   1.0,   20 },   -- 原 1→10
{ 100,  300,    20,   60 },   -- 原 10→30
```

方案 B — 推迟进阶/升星解锁时间。

### 建议 4: 提前 ENEMY_SCALING 生效节点 (优先级: 🟡中)

```lua
critDmgReduce = {
    {    1,   100,  0.00, 0.05 },   -- 从 100 关开始 (原 300 关)
    {  100,   500,  0.05, 0.15 },
    {  500,  1500,  0.15, 0.25 },
    { 1500,  3000,  0.25, 0.32 },
    { 3000,  6000,  0.32, 0.40 },
}
```

### 建议 5: 提升 SPD 攻速上限 (优先级: 🟢低)

```lua
Config.SPD_BONUS_CURVE = {
    { 1,    0.00 },
    { 3,    0.10 },
    { 10,   0.20 },
    { 50,   0.30 },
    { 200,  0.40 },
    { 1000, 0.50 },   -- 新增
}
Config.SPD_BONUS_MAX = 0.50
```

### 建议 6: 降低 BOSS 后期 HP 指数 (优先级: 🟢低)

BOSS HP = baseHP × roundHPMult × hpScale × BossTier^2.25。
建议降指数从 2.25 到 1.8，或对 BossTier 增加上限。

## 六、总结

| 优先级 | 问题 | 建议 |
|--------|------|------|
| 🔴 高 | 3000关后英雄追不上怪物HP | 压平后半段HP曲线 |
| 🔴 高 | 满星满阶后无养成空间 | 增加超越/转生系统 |
| 🟡 中 | 前500关体验无挑战 | 提高前期怪物基础HP |
| 🟡 中 | 防御缩放生效太晚 | 提前ENEMY_SCALING生效节点 |
| 🟢 低 | 攻速上限30%太低 | 提升至50% |
| 🟢 低 | BOSS后期HP过于离谱 | 降低BossTier指数 |

**核心结论**: 英雄的增长是有限的乘法（等级线性 × 进阶20阶 × 升星30星 = 有上限），
而怪物HP是无限的分段加速增长（3500万倍 >> 63万倍）。
要么压低怪物增长率使两条曲线在终局相交，要么给英雄增加新的无上限养成维度。
